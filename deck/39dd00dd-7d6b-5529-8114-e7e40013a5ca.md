<front>

Is it required to run `dbt compile` immediately before running `dbt run` to ensure the code is up to date?

</front>

---

<back>

No. `dbt run` automatically performs the compilation step as part of its execution lifecycle before materializing models. `dbt compile` is primarily used for debugging and manual inspection of generated SQL.

</back>
