<front>

A developer configures a Python model with `dbt.config(materialized='view')`. What will happen when `dbt run` is executed?

</front>

---

<back>

dbt will return an error. Python models cannot be materialized as views because dbt cannot compile arbitrary Python code into a SQL view definition.

</back>
