<front>

How does the dbt Cloud IDE 'Preview' button handle result set sizes by default?

</front>

---

<back>

The IDE automatically appends a **limit clause** (defaulting to 500 rows) to the query to prevent browser crashes. If you want to override this, you must write an explicit `LIMIT` clause in your SQL file.

</back>
