<front>

You are creating a high-volume table on Databricks and want to use Z-Ordering to improve data skipping on the `event_time` column. How do you configure this?

</front>

---

<back>

Add the `zorder` parameter to the model config.

```sql
{{ config(
  materialized='incremental',
  zorder=["event_time"]
) }}
```
This instructs the adapter to run the optimization command after table creation.

</back>
