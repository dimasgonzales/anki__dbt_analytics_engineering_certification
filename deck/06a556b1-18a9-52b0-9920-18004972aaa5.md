<front>

What is the primary intended use case for the `dbt docs serve` command?

</front>

---

<back>

The `dbt docs serve` command is designed strictly for **local development** and previewing.

It spins up a lightweight local web server (defaulting to port 8080) to view the documentation but lacks the security and performance features required for production hosting.

</back>
