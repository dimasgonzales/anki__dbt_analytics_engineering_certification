<front>

What are the five valid values allowed for the mandatory `type` property in a dbt exposure definition?

</front>

---

<back>

The valid types are:
1. `dashboard`
2. `notebook`
3. `analysis`
4. `ml`
5. `application`

Any other value will cause a validation error during parsing.

</back>
