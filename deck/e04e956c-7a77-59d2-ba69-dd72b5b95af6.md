<front>

Which configuration key in `dbt_project.yml` tells dbt where to look for CSV files to load?

</front>

---

<back>

The `seed-paths` configuration key defines the directory (or directories) where dbt looks for seed files.

```yaml
# dbt_project.yml
seed-paths: ["seeds"]
```

</back>
