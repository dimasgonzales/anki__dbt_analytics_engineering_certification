<front>

Your CI/CD pipeline is failing because the linter is flagging syntax errors in your snapshot files. What is the best practice to resolve this without modifying the snapshot logic?

</front>

---

<back>

Add the `snapshots/` directory to a `.sqlfluffignore` file. Snapshots often contain specialized logic that does not conform to standard SQL linting rules, so they should be explicitly excluded to avoid false positives.

</back>
