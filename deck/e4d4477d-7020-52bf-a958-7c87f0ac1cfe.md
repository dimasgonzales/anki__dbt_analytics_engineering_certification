You are updating the `monthly_revenue` model. Before running it, you need to ensure all its source tables and intermediate calculations are refreshed. Which command should you run?

</front>

---

<back>

You should run:

```bash
dbt run --select +monthly_revenue
```

Putting the `+` **before** the model name selects the model and all its upstream ancestors, ensuring inputs are fresh.

</back>
