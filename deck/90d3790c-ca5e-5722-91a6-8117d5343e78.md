<front>

You have tagged specific sensitive models with `meta: {contains_pii: true}` in their YAML configuration. How would you list all these models using the CLI?

</front>

---

<back>

You can target nested configuration keys using dot notation within the `config:` selector.

```bash
dbt list --select "config.meta.contains_pii:true"
```

</back>
