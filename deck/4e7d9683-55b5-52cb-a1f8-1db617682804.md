<front>

Your governance team requires the "Data Steward" email to be visible for every source table in the dbt documentation site. How do you accomplish this?

</front>

---

<back>

Add the `meta` key to the source or table configuration with the required field:

```yaml
sources:
  - name: stripe
    tables:
      - name: payments
        meta:
          data_steward: "finance@example.com"
```

</back>
