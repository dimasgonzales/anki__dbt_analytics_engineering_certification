<front>

You are developing in the dbt Cloud IDE and want to manually trigger a seed load for testing. How do you accomplish this?

</front>

---

<back>

You can execute the command using the embedded command bar usually located at the bottom of the IDE interface.

Example input:
```bash
dbt seed
```
Alternatively, you can use the interface buttons (e.g., "Run") if configured for that specific action.

</back>
