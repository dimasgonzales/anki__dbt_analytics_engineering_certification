<front>

You are developing complex logic and want to run only your unit tests to check the logic against mock data, avoiding the overhead of running data tests against the warehouse. How do you accomplish this?

</front>

---

<back>

Use the `test_type` selector specifying `unit`.

```bash
dbt test --select "test_type:unit"
```

</back>
