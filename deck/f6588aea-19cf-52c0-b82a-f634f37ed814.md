<front>

You are developing a complex Jinja macro and want to verify the rendered SQL for a specific logic snippet without running a full model or saving a file. How can you do this efficiently?

</front>

---

<back>

Use the `--inline` flag with `dbt compile` to print the compiled SQL directly to the terminal. 

Example:
```bash
dbt compile --inline "select * from {{ ref('raw_orders') }}"
```

</back>
