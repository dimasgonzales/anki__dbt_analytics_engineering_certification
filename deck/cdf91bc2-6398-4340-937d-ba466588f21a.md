What is the environment variable order of precedence in dbt Cloud?
</front>

---

<back>

1. Personal/Job override
2. Environment
3. Project default
4. Jinja function default arg (i.e. the second argument in `env_var('DBT_MY_VAR', 'default')`;

</back>
