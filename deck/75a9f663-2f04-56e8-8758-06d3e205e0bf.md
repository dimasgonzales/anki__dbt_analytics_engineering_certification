<front>

Where can a developer find the fully resolved SQL code that dbt generates from a model containing Jinja macros?

</front>

---

<back>

In the `target/compiled/` directory. Before execution, dbt compiles all Jinja logic into pure SQL and saves it here. This allows developers to see exactly what query is being sent to the data warehouse.

</back>
