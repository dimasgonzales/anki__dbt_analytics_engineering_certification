<front>

Since Jinja is not rendered inside Python model code, how do you access project variables defined in `dbt_project.yml`?

</front>

---

<back>

You use the `dbt.config.get()` method to access variables passed into the configuration context.

```python
my_var = dbt.config.get('my_variable_name')
```

</back>
