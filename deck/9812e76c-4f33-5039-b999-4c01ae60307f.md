<front>

You define an incremental model with `incremental_strategy='merge'` but receive an error. You are using the default table type (Hive). What is the cause?

</front>

---

<back>

The `merge` strategy is exclusive to `table_type='iceberg'`. To fix this, you must change the table type and ensure a `unique_key` is defined.
```sql
{{ config(
  table_type='iceberg', 
  incremental_strategy='merge', 
  unique_key='id'
) }}
```

</back>
