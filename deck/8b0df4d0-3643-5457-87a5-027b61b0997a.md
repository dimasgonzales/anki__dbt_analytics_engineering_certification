<front>

You are building a model downstream of a snapshot and need to select only the current state of the data. How would you filter the query?

</front>

---

<back>

Filter for rows where `dbt_valid_to` is `null`.
```sql
select *
from {{ ref('my_snapshot') }}
where dbt_valid_to is null
```

</back>
