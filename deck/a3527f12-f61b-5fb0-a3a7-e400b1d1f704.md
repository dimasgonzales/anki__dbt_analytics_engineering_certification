<front>

What is the difference between using the **`+` operator** and the **`@` operator** in dbt node selection syntax?


</front>

---

<back>

These operators are used with a node's name (e.g., `model_a`) to select related nodes:

* **`+` Operator (Ancestors/Descendants):** Selects dependencies either upstream or downstream.
    * **`+model_a`**: Selects the node (`model_a`) and **all its upstream parents** (ancestors).
    * **`model_a+`**: Selects the node (`model_a`) and **all its downstream children** (descendants).

* **`@` Operator (The Build Set):** Selects the complete set of nodes required for a safe build.
    * **`@model_a`**: Selects the node, its parents, its children, **and the parents of those children**.


</back>
