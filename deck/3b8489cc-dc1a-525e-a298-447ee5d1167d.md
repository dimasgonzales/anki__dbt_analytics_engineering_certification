<front>

What does the `{{ source() }}` function actually render into the compiled SQL file that executes against the warehouse?

</front>

---

<back>

It renders the **fully qualified relation name** of the object (e.g., `database.schema.table`). This creates a valid SQL reference that points to the physical location of the data in the target data warehouse.

</back>
