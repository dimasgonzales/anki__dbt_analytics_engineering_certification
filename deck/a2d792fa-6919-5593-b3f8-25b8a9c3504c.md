<front>

What is the primary purpose of the `package-lock.yml` file in dbt?

</front>

---

<back>

It records the exact resolved versions and commit SHAs of all dependencies. This ensures deterministic builds by guaranteeing that the code running in production is identical to what was resolved and tested in development.

</back>
