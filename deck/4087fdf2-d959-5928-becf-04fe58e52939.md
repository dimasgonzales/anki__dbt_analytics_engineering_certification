<front>

How do you invoke a specific macro from the command line in dbt, and how do you pass arguments to it?

</front>

---

<back>

Use the `dbt run-operation` command followed by the macro name. Pass arguments using the `--args` flag formatted as a YAML string.

Example:
```bash
dbt run-operation my_macro --args '{arg_1: value}'


# Example 1
dbt run-operation grant_select --args '{role: reporter}'

# Example 2
dbt run-operation clean_stale_models --args '{days: 7, dry_run: True}'
```

</back>
