You need to configure a singular data test (`tests/assert_total_positive.sql`) to have a severity of `warn`. How do you apply this?

</front>

---

<back>

Add a config block at the top of the SQL file:

```sql
{{ config(severity = 'warn') }}
select ...
```

</back>
