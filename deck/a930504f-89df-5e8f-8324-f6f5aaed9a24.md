<front>

You have defined a macro named `archive_cleanup` to delete old records. You want to trigger this logic immediately without triggering a full `dbt run`. Which command should you use?

</front>

---

<back>

Use `dbt run-operation` with the macro name.

Command:
```bash
dbt run-operation archive_cleanup
```

</back>
