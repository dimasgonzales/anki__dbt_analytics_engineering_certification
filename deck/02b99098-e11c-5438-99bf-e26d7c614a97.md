<front>

What is the primary resource for identifying compatible package versions and usage documentation when resolving dependency errors?

</front>

---

<back>

The **dbt Hub** (hub.getdbt.com). It is the official registry for dbt packages, allowing you to find valid version numbers, read installation instructions, and verify compatibility with your version of dbt Core.

</back>
