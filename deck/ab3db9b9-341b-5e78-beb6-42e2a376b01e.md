<front>

You are migrating a complex transformation from a SQL stored procedure to a dbt Python model. The source data is defined in an existing dbt SQL model named `stg_users`. How do you access this data in your new Python file?

</front>

---

<back>

Call `dbt.ref("stg_users")` inside your Python `model` function. This reads the data from the SQL model into a DataFrame, allowing you to mix SQL preparation with Python transformation while maintaining lineage.

</back>
