<front>

Which three YAML keys are primarily used to configure and manage Model Versioning in dbt?

</front>

---

<back>

1. `versions`: A list defining the specific versions (e.g., `v: 1`, `v: 2`).
2. `latest_version`: Specifies which version is the default when `ref()` is used.
3. `deprecation_date`: (Optional) Marks when a specific version will become invalid.

</back>
