<front>

You have a test on a legacy table that always has a few duplicates. You want the test to error only if there are more than 50 failing rows. How would you configure this in YAML?

</front>

---

<back>

Set the `error_if` config to `">50"`:

```yaml
- unique:
    config:
      error_if: ">50"
```

</back>
