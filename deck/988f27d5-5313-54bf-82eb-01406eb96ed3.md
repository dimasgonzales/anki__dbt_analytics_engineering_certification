<front>

In a dbt source definition, what is the difference between the `name` property and the `identifier` property?

</front>

---

<back>

The `name` property defines how you reference the table within dbt (e.g., `{{ source('src', 'name') }}`), while the `identifier` property corresponds to the actual physical table name in the data warehouse. If omitted, the `identifier` defaults to the `name`.

</back>
