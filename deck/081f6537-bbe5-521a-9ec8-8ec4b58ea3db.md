<front>

How does dbt determine which connection credentials to use when running a project?

</front>

---

<back>

dbt looks at the `profile` key in the `dbt_project.yml` file and searches for an exact matching top-level key in your `profiles.yml` file. If the names match, dbt connects using the credentials defined in the `target` for that profile.

</back>
