<front>

What are the limitations on the number of unique environment types (Production, Staging, Development, General) allowed within a single dbt Cloud project?

</front>

---

<back>

A dbt Cloud project is limited to exactly **one** Production, **one** Staging, and **one** Development environment. However, there is **no limit** on the number of 'General' environments, which can be used for various ad-hoc or mixed-use cases.

</back>
