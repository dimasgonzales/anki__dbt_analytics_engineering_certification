<front>

How do you reference a public model named `fct_orders` located in a separate upstream project named `global_sales`?

</front>

---

<back>

Use the two-argument version of the `ref()` function, specifying the project name first.

```sql
select *
from {{ ref('global_sales', 'fct_orders') }}
```

</back>
