<front>

Which dbt command is strictly required to populate column-level details, descriptions, and statistics in the generated documentation site?

</front>

---

<back>

```bash
dbt docs generate
```
While `dbt run` generates the manifest, `dbt docs generate` queries the information schema to build the `catalog.json` file, which contains the detailed metadata.

</back>
