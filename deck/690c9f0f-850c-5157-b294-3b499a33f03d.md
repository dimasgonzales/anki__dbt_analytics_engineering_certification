<front>

What is the primary function of the `dbt seed` command?

</front>

---

<back>

The `dbt seed` command loads static CSV files from the dbt project into the data warehouse as tables. Unlike `dbt run`, which executes SQL transformations, `dbt seed` parses the CSVs and performs an INSERT operation to populate the tables.

</back>
