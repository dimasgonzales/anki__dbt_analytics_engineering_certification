<front>

You are creating a custom `ref` override for a blue/green deployment strategy. You need the resulting SQL to omit the database name and only render `schema.table`. How do you accomplish this?

</front>

---

<back>

Use the `include` method on the relation object with `database=false`.

```sql
{% macro ref(model_name) %}
  {% set rel = builtins.ref(model_name) %}
  {{ return(rel.include(database=false)) }}
{% endmacro %}
```

</back>
