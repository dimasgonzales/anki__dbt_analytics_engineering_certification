<front>

How do you prevent SQLFluff from analyzing specific directories, such as `snapshots/`, within a dbt project?

</front>

---

<back>

Create a `.sqlfluffignore` file in the project root and list the directories to exclude. dbt does not automatically ignore these folders for linting purposes.

Example content:
```text
snapshots/
```

</back>
