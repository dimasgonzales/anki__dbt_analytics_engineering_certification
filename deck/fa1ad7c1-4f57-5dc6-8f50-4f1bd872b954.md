<front>

A production job seems to be using stale data artifacts despite recent updates, and you need to ensure a completely clean run for troubleshooting. How do you accomplish this?

</front>

---

<back>

Navigate to **Orchestration > Environments > [Select Environment]** and click **Clear cache**. This removes the saved state history, forcing the next job to fully rebuild the cache and all models.

</back>
