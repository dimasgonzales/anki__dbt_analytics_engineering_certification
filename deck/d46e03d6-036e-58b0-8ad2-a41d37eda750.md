<front>

How do you reference an upstream model to establish lineage within a dbt Python model?

</front>

---

<back>

Use the `dbt.ref()` method. This function registers the dependency in the DAG and returns a reference to the upstream dataset (a relation or DataFrame-like object, depending on the adapter).

Example:
```python
my_upstream_df = dbt.ref("stg_customers")
```

</back>
