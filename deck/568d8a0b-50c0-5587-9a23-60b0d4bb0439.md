<front>

You want to ensure downstream dashboards automatically refresh in your Production environment but remain static in your Development environment to save resources. How do you accomplish this?

</front>

---

<back>

Set the environment variable `DBT_ACTIVE_EXPOSURES` to `1` only within the **Production Environment** settings in dbt Cloud. Leave the variable unset or set to `0` in the Development Environment.

</back>
