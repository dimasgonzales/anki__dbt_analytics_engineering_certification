<front>

What is the correct CLI syntax to explicitly disable a boolean flag in dbt?

</front>

---

<back>

Use the `--no-` prefix followed by the flag name. This explicitly sets the boolean value to false.

Example:
```bash
dbt run --no-version-check
```

</back>
