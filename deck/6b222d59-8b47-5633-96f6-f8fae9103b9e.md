<front>

Your Python model performs a heavy machine learning training job and fails after 5 minutes due to a platform limit. How do you extend the allowed execution time for just this model?

</front>

---

<back>

Configure the `timeout` parameter specifically for that model using `dbt.config()` within the file. 

```python
def model(dbt, session):
    # Sets timeout to 20 minutes, value is defined in seconds.
    dbt.config(timeout=1200)
    # ...
```

</back>
