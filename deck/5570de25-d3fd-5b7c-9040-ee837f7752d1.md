<front>

A model fails with a 'Database Error' indicating a syntax issue (e.g., 'SQL compilation error: syntax error line 14'). How should you efficiently troubleshoot this?

</front>

---

<back>

Locate the generated SQL file in `target/run/{project_name}/models/...`. Copy the content into your data warehouse's query console (e.g., Snowflake/BigQuery UI) and execute it manually to isolate and fix the specific syntax error.

</back>
