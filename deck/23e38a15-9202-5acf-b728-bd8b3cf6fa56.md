<front>

Which configuration file must be used when defining private git packages that rely on git token authentication via environment variables?

</front>

---

<back>

You must use `packages.yml`. 

Unlike `dependencies.yml`, `packages.yml` supports Jinja templating, which is required to render functions like `{{ env_var('GITHUB_TOKEN') }}` for passing credentials securely.

</back>
