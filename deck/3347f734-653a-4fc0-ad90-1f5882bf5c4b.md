<front>
How can you configure a specific resource to be ignored by dbt?
</front>

---

<back>

Using the `enabled` config. Setting `enabled: false` causes dbt to completely ignore the resource, even if it causes compilation errors (the default for all resources is `enabled: true`)

</back>
