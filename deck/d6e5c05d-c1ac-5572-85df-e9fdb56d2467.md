<front>

What file must exist in the current directory for dbt CLI commands to execute successfully?

</front>

---

<back>

The `dbt_project.yml` file.

dbt commands must generally be run from the project's root directory because dbt looks for this file to determine the project name, profile configuration, and model paths.

</back>
