<front>

What is the syntax for defining model configurations directly inside a SQL model file?

</front>

---

<back>

Use the `{{ config() }}` Jinja macro, typically placed at the top of the file. It accepts arguments as key-value pairs.

```sql
{{ config(
    materialized='table',
    schema='marketing'
) }}
```

</back>
