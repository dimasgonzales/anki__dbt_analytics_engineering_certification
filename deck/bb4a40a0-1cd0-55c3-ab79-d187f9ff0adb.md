<front>

Which dbt command allows you to see the resolved SQL for a file in the `analyses/` directory, and where is the output located?

</front>

---

<back>

Run `dbt compile`. The resolved SQL (with Jinja replaced by actual SQL) is generated in the `target/` directory path:
```bash
target/compiled/{project_name}/analyses/{filename}.sql
```

</back>
