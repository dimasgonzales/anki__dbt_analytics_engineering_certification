How do you configure specific warehouse connection parameters or driver options that are not explicitly provided in the dbt Cloud UI fields?

</front>

---

<back>

Use the 'Extended Attributes' configuration section. You can provide these settings as key-value pairs (often matching `profiles.yml` syntax) to pass specific driver options directly to the adapter.

</back>
