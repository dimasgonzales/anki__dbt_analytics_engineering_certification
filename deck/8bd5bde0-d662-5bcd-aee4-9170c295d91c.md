What is the required function signature for defining a dbt Python model?

</front>

---

<back>

A dbt Python model must define a function named `model` that accepts exactly two arguments: `dbt` and `session`. This function must return a DataFrame to be materialized.

```python
def model(dbt, session):
    # ... logic ...
    return final_df
```

</back>
