<front>

In the dbt Cloud Audit Log, what is the difference between `actor_type: User` and `actor_type: Service`?

</front>

---

<back>

*   `User`: Indicates an action performed by a specific person logged into the UI.
*   `Service`: Indicates an action performed programmatically via an API request or Service Token.

</back>
