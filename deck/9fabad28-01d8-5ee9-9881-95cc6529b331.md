<front>

You created a new lightweight job specifically to update documentation every hour. However, the 'Documentation' link in dbt Cloud is still showing stale data from your daily production job. How do you fix this?

</front>

---

<back>

You need to update the Project Settings. Go to **Account Settings -> Projects -> [Project Name] -> Edit -> Artifacts**, and change the selection to point to your new hourly documentation job ID.

</back>
