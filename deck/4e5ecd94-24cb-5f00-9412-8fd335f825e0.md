<front>

What function is used within a `schema.yml` file to insert the content of a defined documentation block?

</front>

---

<back>

The `doc()` function is used. It renders the contents of the specified documentation block into the YAML file.

Example:
```yaml
description: "{{ doc('my_doc_block') }}"
```

</back>
