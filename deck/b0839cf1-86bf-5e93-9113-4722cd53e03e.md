<front>

You are using a Jinja `for` loop to `UNION ALL` a list of tables. How do you ensure the `UNION ALL` keyword does not appear after the final table?

</front>

---

<back>

Use the `loop.last` variable to conditionally check if the current iteration is NOT the last one.

Example:
```sql
{% for table in tables %}
  SELECT * FROM {{ table }}
  {% if not loop.last %} UNION ALL {% endif %}
{% endfor %}
```

</back>
