<front>

You are troubleshooting a cryptic error during `dbt run` where the terminal output is truncated or insufficient. What is the first step to retrieve the full debug trace?

</front>

---

<back>

Inspect the `logs/dbt.log` file in your project directory.

This file captures detailed runtime events, including `DEBUG` logs and complete SQL error messages that dbt may hide from the standard CLI output to keep it readable.

</back>
