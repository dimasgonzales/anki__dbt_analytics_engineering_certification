<front>

You need to generate a list of all model names and their unique IDs from a specific deployment for a data catalog integration. How would you accomplish this using dbt Cloud?

</front>

---

<back>

You should execute a GraphQL query against the **Metadata API**.

By targeting the specific `environment` and requesting the `lineage` object with a filter for `Model` types, you can programmatically extract the `name` and `uniqueId` for every node in that environment's DAG.

</back>
