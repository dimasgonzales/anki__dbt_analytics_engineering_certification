<front>

How do you explicitly specify which Python version the data platform should use to execute a dbt Python model?

</front>

---

<back>

You use the `python_version` argument within the `dbt.config()` method inside the model file. 

Example:
```python
def model(dbt, session):
    dbt.config(python_version="3.10")
    ...
```

</back>
