<front>

How can you select models that match a specified node config?

</front>

---

<back>

```
--select config.<option path>:<value>
```  
For example:  

- `dbt run --select config.materialized:incremental` runs all models that are materialized incrementally
- `dbt ls -s config.grants.select:reporter` lists all models where the "reporter" entity has SELECT privileges

</back>
