<front>

How does dbt determine the list of columns to generate when using the `dbt_utils.star` macro?

</front>

---

<back>

It uses database introspection during the compilation phase. Helper macros query the information schema (e.g., using `adapter.get_columns_in_relation`) to retrieve the existing column names before the final SQL is compiled.

</back>
