When debugging a complex macro, which three documentation sources correspond to logic errors, context errors, and data object errors respectively?

</front>

---

<back>

1. **Jinja2 Docs**: Logic errors (loops, if-statements, filters).
2. **dbt Docs**: Context errors (config, ref, adapter functions).
3. **Agate Docs**: Data object errors (manipulating query results, accessing rows/columns).

</back>
