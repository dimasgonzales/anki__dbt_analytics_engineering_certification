<front>

Which operator is used to concatenate strings and variables within a dbt `log` function call?

</front>

---

<back>

The tilde operator (`~`) is used for concatenation in Jinja, rather than the SQL pipe (`||`) or Python plus (`+`).

```jinja
{{ log("Variable value: " ~ my_variable, info=True) }}
```

</back>
