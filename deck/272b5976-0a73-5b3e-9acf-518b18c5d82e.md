<front>

You want to execute the data tests defined on your source tables (`unique`, `not_null`, etc.) without running the actual models. Which command should you use?

</front>

---

<back>

Use `dbt test` with the `source:` selector.

```bash
$ dbt test --select source:jaffle_shop
```

</back>
