<front>

You want to enforce custom linting rules in dbt Cloud instead of using the default opinionated formatting. What step must you take to enable this?

</front>

---

<back>

Add a `.sqlfluff` file to your project root. This overrides the default `sqlfmt` behavior, allowing you to define custom rules and perform deep code analysis.

Note: You can also use nested `.sqlfluff` files for folder-specific rules.

</back>
