<front>

You are building an incremental model and need to define the `unique_key` for row updates. How would you configure this within the model file itself?

</front>

---

<back>

Add the `unique_key` argument to the `config` macro inside the model's SQL file.

```sql
{{ config(
    materialized='incremental',
    unique_key='transaction_id'
) }}

SELECT ...
```

</back>
