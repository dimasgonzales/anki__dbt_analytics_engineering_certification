<front>

What **dbt\_project.yml** field is used to specify a custom list of directories where data tests are located? What is the default?

</front>

---

<back>

**test-paths**. Default is `tests`. Specifically, for each directory specified in `test-paths`, dbt will assume any top-level SQL files are **singular** tests, and any SQL files in the `/generic` subdirectory are **generic** tests.

</back>
