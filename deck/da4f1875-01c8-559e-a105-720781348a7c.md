<front>

How do you configure dbt snapshots to use custom names for system-generated columns like `dbt_valid_from`?

</front>

---

<back>

You can use the `snapshot_meta_column_names` configuration in `dbt_project.yml` or within a snapshot's config block. This accepts a dictionary mapping the default dbt names to your desired column names.

```yaml
config:
  snapshot_meta_column_names:
    dbt_valid_from: effective_from
```

</back>
