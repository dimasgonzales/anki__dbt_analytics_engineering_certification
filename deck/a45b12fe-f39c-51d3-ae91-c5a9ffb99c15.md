<front>

Where does the dbt Cloud CLI store authentication context and active project settings, and where is this file located?

</front>

---

<back>

It stores these settings in `dbt_cloud.yml`, located in the `~/.dbt/` directory. Unlike dbt Core, the Cloud CLI does not rely primarily on `profiles.yml` for connection details, as these are managed via the dbt Cloud platform integration.

</back>
