<front>

How does dbt compile a model configured with `materialized='ephemeral'`, and what is the primary impact on the data warehouse?

</front>

---

<back>

dbt compiles ephemeral models as Common Table Expressions (CTEs) interpolated directly into downstream models. No actual table or view is created in the data warehouse, which helps keep the schema clean but prevents direct querying of the ephemeral model.

</back>
