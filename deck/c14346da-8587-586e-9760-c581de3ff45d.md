<front>

How can you set a default `severity` of `warn` for **all** tests in a project without modifying individual model files?

</front>

---

<back>

Configure the `data_tests` key in `dbt_project.yml` using the `+` prefix:

```yaml
data_tests:
  +severity: warn
```

</back>
