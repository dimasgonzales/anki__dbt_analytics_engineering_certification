<front>

Which Jinja context variables and functions are strictly allowed when defining resource configurations in `dbt_project.yml`?

</front>

---

<back>

You are limited to using `env_var`, `var`, `target`, and `builtins`. You **cannot** use custom macros, `ref()`, or `source()` within the `dbt_project.yml` configuration blocks.

</back>
