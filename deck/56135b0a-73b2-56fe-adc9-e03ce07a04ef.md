<front>

How do you enable contract enforcement for a specific model in dbt?

</front>

---

<back>

You must add the `contract` configuration with `enforced: true` to the model's properties in the YAML file.

Example:
```yaml
models:
  - name: my_model
    config:
      contract:
        enforced: true
```

</back>
