<front>

You are debugging a compilation error in `generate_schema_name`. Which of the following variables is safe to use: `target.name` or the result of a SQL query?

</front>

---

<back>

`target.name` is safe to use.

This macro operates with a limited Jinja context. While configuration variables like `target` and `env_var` are available, dynamic SQL results are not accessible during schema generation.

</back>
