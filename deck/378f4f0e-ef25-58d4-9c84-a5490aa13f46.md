<front>

Which built-in dbt macro should be used to add or subtract time from a date field to ensure compatibility across different SQL dialects?

</front>

---

<back>

The `{{ dbt.dateadd() }}` macro. It normalizes the syntax for adding intervals (like days, months, or hours) to a date or timestamp, handling dialect-specific differences (such as `DATEADD` vs `DATE_ADD` or varying interval syntax).

</back>
