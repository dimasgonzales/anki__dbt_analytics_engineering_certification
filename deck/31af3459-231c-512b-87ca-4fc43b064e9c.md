<front>

You are troubleshooting a macro and want to inspect the full structure of the `model` object directly in your IDE's compiled SQL preview. How would you do this?

</front>

---

<back>

Use the `tojson` filter to render the object as a JSON string within the file content.

Example:
```sql
/*
Debug Output:
{{ model | tojson(indent=4) }}
*/
```

</back>
