<front>

How does dbt distinguish between Union (OR) logic and Intersection (AND) logic when providing multiple arguments to the `--select` flag?

</front>

---

<back>

dbt interprets **space-separated** arguments as a **Union (OR)** (selects matches for any argument). It interprets **comma-separated** arguments (without spaces) as an **Intersection (AND)** (selects only items matching all arguments).

Example:
```bash
# Union
dbt run --select tag:A tag:B

# Intersection
dbt run --select tag:A,tag:B
```

</back>
