<front>

Where must dbt Sources be defined and configured within a dbt project?

</front>

---

<back>

Sources must be declared in `.yml` files located within the `models/` directory (or paths defined in `model-paths`). Unlike models, sources cannot be configured in `dbt_project.yml` or using `config()` blocks.

</back>
