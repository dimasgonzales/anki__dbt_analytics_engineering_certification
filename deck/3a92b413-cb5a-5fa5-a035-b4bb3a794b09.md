<front>

One of your incremental models processes a massive amount of data and consistently fails due to a default 5-minute database statement timeout. How do you fix this for just this model?

</front>

---

<back>

Add a `pre_hook` to the model configuration to increase the timeout limit.

```sql
{{
  config(
    materialized='incremental',
    pre_hook="set statement_timeout = '30min'"
  )
}}
```

</back>
