<front>

In `sources.yml`, how does defining a property like `loaded_at_field` at the top source level affect the tables nested underneath it?

</front>

---

<back>

Properties defined at the source level **cascade** down to all tables within that source group. This applies the configuration to every table unless it is explicitly overridden at the individual table level.

</back>
