<front>

What does the `dbt run-operation` command do?

</front>

---

<back>

It is used to invoke a macro from the command line. The full syntax is:  
`dbt run-operation {macro} --args {args}`  
where {args} is a YAML string of key-value pairs matching the keyword arguments of the macro.  
  
E.g. 
```
dbt run-operation grant_select --args '{role: reporter}'
```

</back>
