<front>

What syntax is used to call a macro named `surrogate_key` from the installed `dbt_utils` package?

</front>

---

<back>

You must namespace the macro with the package name (unless dispatched).

Syntax:
```jinja
{{ dbt_utils.surrogate_key(['field_1', 'field_2']) }}
```

</back>
