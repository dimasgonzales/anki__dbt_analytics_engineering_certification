<front>

Which dbt ecosystem package is specifically designed to help with migration projects by comparing the data returned by a legacy SQL query against a new dbt model?

</front>

---

<back>

`dbt_audit_helper`

This package includes macros like `compare_relations` which generate SQL to identify row-by-row differences between two datasets, making it essential for regression testing refactors.

</back>
