<front>

How does `dbt run` handle a selector that matches multiple resource types (e.g., models, seeds, and tests)?

</front>

---

<back>

`dbt run` automatically filters the selected nodes to execute only **models**. It will ignore seeds, tests, and snapshots even if they match the selection criteria.

</back>
