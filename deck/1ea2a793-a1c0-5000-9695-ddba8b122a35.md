<front>

How do you declare upstream dependencies for an `exposure` in dbt, given that exposures do not contain SQL code to parse?

</front>

---

<back>

You must explicitly define dependencies using the `depends_on` property in the YAML configuration. This list can contain calls to `ref()`, `source()`, or `metric()`.

Example:
```yaml
depends_on:
  - ref('fct_orders')
  - source('raw', 'transactions')
```

</back>
