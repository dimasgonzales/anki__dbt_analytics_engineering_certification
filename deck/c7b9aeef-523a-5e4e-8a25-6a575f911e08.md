<front>

You need to ensure a specific large model in Databricks is optimized for queries filtering by `region` and `date`. How would you implement this in the model file?

</front>

---

<back>

Use the `config` block to apply Z-Ordering specific to Databricks within the model definition:
```sql
{{ config(
  materialized='incremental',
  zorder=['region', 'date']
) }}
```

</back>
