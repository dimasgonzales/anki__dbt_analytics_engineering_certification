<front>

What object type does the `run_query` macro return in dbt, and how do you access a specific column's values from it?

</front>

---

<back>

It returns an **Agate table** object. 

You can access a specific column's values using the syntax `results.columns['column_name'].values()` or by index `results.columns[0].values()`.

</back>
