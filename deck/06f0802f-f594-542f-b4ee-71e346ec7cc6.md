<front>

You have two separate dbt Cloud projects that connect to the exact same GitHub repository. How should you handle the Deploy Keys?

</front>

---

<back>

You must add **two distinct Deploy Keys** to the GitHub repository. dbt Cloud generates a unique SSH key for every specific project, and keys are project-scoped. You cannot reuse the generated key from Project A for Project B.

</back>
