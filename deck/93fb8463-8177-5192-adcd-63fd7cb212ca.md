<front>

You observe in dbt Explorer that `orders_snapshot` has increased in runtime from 5 minutes to 45 minutes over two months. What is the recommended action?

</front>

---

<back>

This trend suggests the dataset is growing and the model is performing full scans. Refactor the model configuration to use an **incremental strategy**.

```yaml
{{ config(
    materialized='incremental',
    unique_key='order_id'
) }}
```

</back>
