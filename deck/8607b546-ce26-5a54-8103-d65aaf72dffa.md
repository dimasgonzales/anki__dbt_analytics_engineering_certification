<front>

What condition determines if a dbt data test passes or fails?

</front>

---

<back>

In dbt, a test is a SQL query. The test **passes** if the query returns **zero rows**. The test **fails** if the query returns **one or more rows**. dbt effectively runs a `select count(*)` wrapper around your test query to check this.

</back>
