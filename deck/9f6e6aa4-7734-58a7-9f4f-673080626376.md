<front>

How do you view the specific arguments, flags, and usage instructions for the `dbt run` command directly in the terminal?

</front>

---

<back>

You use the `--help` flag after the specific command.

Example:
```bash
dbt run --help
```
This lists available flags like `--select`, `--full-refresh`, and `--fail-fast` specific to the run command.

</back>
