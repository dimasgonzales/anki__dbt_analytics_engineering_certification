<front>

Which dbt CLI selection syntax allows you to run only the version of a model that is currently configured as the `latest_version`?

</front>

---

<back>

Use the intersection operator `,` with the `version:latest` selector.

Command:
```bash
dbt run --select model_name,version:latest
```

</back>
