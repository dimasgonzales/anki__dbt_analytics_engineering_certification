<front>

How can you conditionally apply model configurations (like materialization) in `dbt_project.yml` based on the active environment?

</front>

---

<back>

You can use the `target` object within inline Jinja expressions. A common pattern checks `target.name`:

```yaml
+materialized: "{{ 'view' if target.name == 'dev' else 'table' }}"
```

</back>
