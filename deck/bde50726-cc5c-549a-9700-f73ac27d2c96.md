<front>

Which specific test from the `dbt_utils` package is designed to validate uniqueness across a list of multiple columns?

</front>

---

<back>

The `dbt_utils.unique_combination_of_columns` test. It accepts a `combination_of_columns` argument listing the fields that form the unique constraint.

```yaml
- dbt_utils.unique_combination_of_columns:
    combination_of_columns:
      - col_a
      - col_b
```

</back>
