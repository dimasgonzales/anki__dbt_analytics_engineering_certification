<front>

What is the `results` context variable in dbt and what data source does it mirror?

</front>

---

<back>

The `results` variable is a list of Result objects available in `on-run-end` hooks that contains execution metadata (status, timing, messages) for every node in the run. It mirrors the data found in the `run_results.json` artifact generated after execution.

</back>
