<front>

Which selector syntax allows you to specifically target and run tests on all defined Sources in your project?

</front>

---

<back>

Use the `source:*` selector:

```bash
dbt test --select "source:*"
```
This selects all source nodes and, via indirect selection, runs the tests attached to them.

</back>
