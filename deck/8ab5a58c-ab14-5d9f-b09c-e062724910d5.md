<front>

How do you explicitly revoke all permissions for a specific privilege on a model using dbt grants?

</front>

---

<back>

You must configure the privilege with an **empty list**.
```yaml
grants:
  select: []
```
This signals dbt to issue revoke statements for that privilege.

</back>
