<front>

What specific docs block name is reserved for overriding the default landing page of the generated dbt documentation site?

</front>

---

<back>

`__overview__`

By defining a docs block with this name, dbt replaces the default "Welcome to your dbt documentation" text with your custom Markdown content.

</back>
