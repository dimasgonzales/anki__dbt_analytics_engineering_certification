<front>

Which configuration dictionary allows you to rename the default meta-columns (like `dbt_valid_from` and `dbt_valid_to`) generated by dbt snapshots?

</front>

---

<back>

Use the `snapshot_meta_column_names` configuration. Within this dictionary, map the internal dbt key (e.g., `dbt_valid_to`) to your desired column name.
```yaml
config:
  snapshot_meta_column_names:
    dbt_valid_to: effective_end_date
```

</back>
