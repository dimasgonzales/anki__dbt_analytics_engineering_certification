<front>

You are creating a new dbt project and want to ensure the configuration file is explicitly defined for forward compatibility. How should you start your `dbt_project.yml`?

</front>

---

<back>

You should explicitly set the config version to 2 at the top of the file.

Example:
```yaml
config-version: 2

name: 'my_project'
version: '1.0.0'
```

</back>
