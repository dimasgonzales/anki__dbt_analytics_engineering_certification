<front>

What is the difference between the `log` and `print` functions?

</front>

---

<back>

- `log` writes to the log file, but *not* to stdout
- *unless* you pass `True` to the optional `info` parameter, in which case `log` is equivalent to `print`
- *unless* the global config `QUIET` is set to `True`, in which case `print` will still produce output but non-error `log` messages will be silenced

</back>
