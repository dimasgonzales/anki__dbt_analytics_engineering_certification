<front>

What configuration property allows you to decouple the logical name of a source table in dbt from its physical table name in the database?

</front>

---

<back>

The `identifier` property. It allows you to assign a clean, user-friendly `name` for use in the `{{ source() }}` function while pointing to the actual, potentially messy physical table name using `identifier`.

</back>
