<front>

You have a maintenance macro named `grant_select` that accepts a `role` argument. How would you execute this from the command line?

</front>

---

<back>

Use `dbt run-operation` and pass arguments using the `--args` flag with a YAML or JSON formatted string.

```bash
dbt run-operation grant_select --args '{role: reporter}'
```

</back>
