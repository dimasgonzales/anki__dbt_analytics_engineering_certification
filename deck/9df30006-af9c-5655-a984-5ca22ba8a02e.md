<front>

What syntax blocks are required to define and strictly close a macro in dbt?

</front>

---

<back>

Macros must be defined using the `{% macro %}` block and closed with `{% endmacro %}`. 

The body contains the reusable SQL or Jinja logic.

Example:
```sql
{% macro my_macro(arg) %}
  ...logic...
{% endmacro %}
```

</back>
