<front>

Your team organizes raw CSV files in a folder named `inputs` instead of the default `seeds` directory. How do you ensure dbt recognizes these files as seeds?

</front>

---

<back>

You must update the `seed-paths` configuration in your `dbt_project.yml` file to point to the custom directory.

```yaml
seed-paths: ["inputs"]
```

</back>
