<front>

You need to verify that an incremental model correctly processes new data in CI without rebuilding the entire history. How would you accomplish this using CLI commands?

</front>

---

<back>

First, clone the existing production model to the CI target, then run the incremental update:

```bash
# 1. Create baseline from prod
dbt clone --select my_model --target ci

# 2. Run incremental logic
dbt run --select my_model --target ci
```

</back>
