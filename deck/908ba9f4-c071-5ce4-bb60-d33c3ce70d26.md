<front>

How do you define database indexes within a dbt model configuration, and what is the mandatory key for each index definition?

</front>

---

<back>

You define indexes using the `indexes` config key, which accepts a list of dictionaries. The mandatory key for each dictionary is `columns` (a list of column names). Optional keys include `unique` and `type`.

```sql
{{ config(
    indexes=[{'columns': ['id']}]
) }}
```

</back>
