<front>

What node selection syntax is used to run only models that have changed compared to a reference state, plus their downstream dependencies?

</front>

---

<back>

The `state:modified+` selector.

This syntax relies on comparing the current project state against a manifest artifact from a previous run (usually production) to identify modified nodes.

</back>
