<front>

How does dbt determine the Directed Acyclic Graph (DAG) and dependencies between models?

</front>

---

<back>

dbt parses the project's SQL files and looks for calls to the `ref()` function (for models/seeds) and `source()` function. It uses these references to construct the dependency graph, determining the order in which models must be executed.

</back>
