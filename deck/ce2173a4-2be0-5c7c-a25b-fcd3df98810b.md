<front>

You want to ensure that ONLY a specific model (`complex_forecast.py`) runs via BigFrames, while the rest of your project defaults to a different method. How do you configure this?

</front>

---

<back>

Apply the configuration directly within the Python model file using the `dbt.config()` method.

Example:
```python
def model(dbt, session):
    dbt.config(submission_method="bigframes")
    ...
```

</back>
