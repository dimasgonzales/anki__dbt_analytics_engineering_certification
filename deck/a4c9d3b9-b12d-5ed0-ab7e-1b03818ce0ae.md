<front>

Which key must be used in `packages.yml` to install a package from a specific directory on the file system rather than from a git repository or the dbt Hub?

</front>

---

<back>

The `local` key.

It specifies the file path to the package, which is commonly used for integration testing a package while developing it.

Example:
```yaml
packages:
  - local: ../
```

</back>
