<front>

In the dbt Cloud IDE, how can you visually verify that a file is correctly being ignored by your `.gitignore` configuration?

</front>

---

<back>

Check the file explorer pane in the IDE. Files and directories that are successfully ignored by git will have their filenames displayed in ***italics***.

</back>
