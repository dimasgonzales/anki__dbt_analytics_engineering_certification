<front>

In which specific dbt environments is the `--version` flag NOT supported?

</front>

---

<back>

The `--version` flag is not supported within the **dbt Cloud IDE** or as a step in **dbt Cloud Production Jobs**. It is a tool designed specifically for the Command Line Interface (local Core or Cloud CLI) to diagnose environment setup.

</back>
