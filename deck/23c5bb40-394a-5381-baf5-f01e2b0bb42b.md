<front>

What **dbt\_project.yml** field specifies the syntax version being used?

</front>

---

<back>

The field is **`config-version`**.

* It should be set to `config-version: 2`.
* Without explicitly specifying the version, dbt **defaults** to using **version 2 syntax**.
* Version 1 syntax was deprecated in dbt v0.19.0.


</back>
