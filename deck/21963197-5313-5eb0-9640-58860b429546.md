<front>

How can you force a resource to always (or never) do a full refresh when run?

</front>

---

<back>

Using the `full_refresh: true|false` config. If this config is set, the model will always follow the specified behavior, regardless of whether the `--full-refresh` flag is invoked in the CLI

</back>
