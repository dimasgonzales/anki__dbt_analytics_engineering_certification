<front>

Which macro from the `dbt_utils` package is standard for obtaining a list of distinct values from a database column for dynamic pivoting?

</front>

---

<back>

`dbt_utils.get_column_values`.

This macro handles the introspection logic automatically, including the `run_query` execution, the `execute` check for parsing safety, and the extraction of values from the resulting Agate table.

</back>
