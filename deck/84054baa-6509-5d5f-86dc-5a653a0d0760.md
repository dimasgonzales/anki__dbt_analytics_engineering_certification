<front>

What is the purpose of the `--inline` flag when using `dbt show`, and what is the default row limit for the output?

</front>

---

<back>

The `--inline` flag allows you to execute an ad-hoc SQL string that leverages dbt's Jinja context (like `{{ ref() }}`) without creating a file. The default output limit is **5 rows**.

</back>
