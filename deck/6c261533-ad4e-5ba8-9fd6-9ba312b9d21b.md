<front>

You have a variable `start_date` defined in your `dbt_project.yml` with a default value of '2023-01-01'. You need to run a backfill for '2020-01-01'. How would you accomplish this via the CLI?

</front>

---

<back>

Use the `--vars` flag in the command line to override the project-level default.

The CLI argument takes precedence over the YAML configuration.

Example:
```bash
dbt run --vars '{"start_date": "2020-01-01"}'
```

</back>
