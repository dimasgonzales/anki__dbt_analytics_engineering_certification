<front>

What does a "Runtime Error: Invalid config version" typically indicate when it references a dependency's `dbt_project.yml` file?

</front>

---

<back>

It indicates that the installed package version is outdated and incompatible with the version of dbt Core you are running. Newer versions of dbt Core often require updated configuration syntax (config version 2) that older packages do not possess.

</back>
