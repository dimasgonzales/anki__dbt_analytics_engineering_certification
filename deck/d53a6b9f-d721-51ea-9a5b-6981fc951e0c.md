<front>

Which git connection types in dbt Cloud support native 'Run on Pull Request' CI triggers?

</front>

---

<back>

Native integrations only (**GitHub, GitLab, and Azure DevOps**). Generic 'Git clone' connections do not support native automated CI triggers within the dbt Cloud UI settings.

</back>
