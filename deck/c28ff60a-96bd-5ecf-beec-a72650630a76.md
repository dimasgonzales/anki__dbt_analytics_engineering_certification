<front>

What is the primary difference between using `| upper` and `is_incremental()` within a dbt model file?

</front>

---

<back>

`| upper` is a **native Jinja filter**, documented in the Jinja2 Template Designer docs, used for string manipulation. `is_incremental()` is a **dbt context function**, specific to the dbt proprietary framework, used to control model materialization logic.

</back>
