<front>

How can you update one table's column sizes to match another table's column sizes? What column types does this work on?

</front>

---

<back>

```
adapter.expand_target_column_types(from_table, to_table)
```  
This only works on **string** and **numeric** columns. A typical use case would be something like expanding a VARCHAR(16) column in the `to_table` to VARCHAR(32) to match the same column in the 
```
from_table
```

</back>
