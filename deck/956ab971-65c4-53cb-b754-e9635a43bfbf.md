<front>

What specific model configuration is required to use the `insert_overwrite` incremental strategy in BigQuery?

</front>

---

<back>

The model must have a `partition_by` configuration. The `insert_overwrite` strategy works by dropping and replacing specific partitions, so dbt must know which column defines the partition boundaries.

</back>
