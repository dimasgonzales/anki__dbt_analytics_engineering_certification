<front>

When defining a custom schema in `dbt_project.yml` for a specific directory, what syntax is required for the key?

</front>

---

<back>

You must use the `+` prefix before the word `schema`.

Example:
```yaml
models:
  project:
    folder:
      +schema: custom_name
```

</back>
