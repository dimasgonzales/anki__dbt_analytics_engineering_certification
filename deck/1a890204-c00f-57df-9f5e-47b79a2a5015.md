What are the three primary use cases where dbt recommends using Python models instead of SQL?

</front>

---

<back>

1. **Machine Learning:** Training models or running predictions.
2. **Complex Math:** Advanced statistics or matrix operations difficult in SQL.
3. **Library Usage:** Parsing difficult data formats using libraries like `dateutil` or `numpy`.
