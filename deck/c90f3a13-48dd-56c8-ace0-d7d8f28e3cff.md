<front>

You need to refresh the data in `status_mappings.csv` and immediately update all models that reference it. What CLI command would you use?

</front>

---

<back>

Use `dbt build` (or run/seed) with the selection syntax including the plus `+` sign for downstream dependencies:
```bash
dbt build --select status_mappings+
```

</back>
