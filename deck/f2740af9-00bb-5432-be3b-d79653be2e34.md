<front>

Which dbt Cloud CLI command is used to automatically reformat SQL files according to configured linting rules?

</front>

---

<back>

The command `dbt sqlfluff format` is used. This utilizes the integrated SQLFluff tool to apply formatting fixes directly to the files based on your configuration.

```bash
dbt sqlfluff format
```

</back>
