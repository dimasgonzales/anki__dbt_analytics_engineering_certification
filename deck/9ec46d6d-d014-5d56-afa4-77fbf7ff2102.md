<front>

Which arguments are required for the `source()` function, and to what configuration do they correspond?

</front>

---

<back>

The `source()` function requires exactly two arguments: `source_name` and `table_name`. These must strictly match the string values of the `name` keys defined in your `sources` YAML configuration (e.g., `src.yml`).

</back>
