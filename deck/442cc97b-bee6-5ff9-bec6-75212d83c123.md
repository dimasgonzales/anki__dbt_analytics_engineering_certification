<front>

You need to configure a specific Python model to use the `table` materialization. How would you apply this configuration within the Python file?

</front>

---

<back>

Call the `dbt.config()` method at the beginning of your `model` function and pass the `materialized` argument.

Example:
```python
def model(dbt, session):
    dbt.config(materialized="table")
    # ...
```

</back>
