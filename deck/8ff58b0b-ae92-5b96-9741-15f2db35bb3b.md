<front>

What CLI command allows you to programmatically add a dependency to `packages.yml` and install it without manually editing the file?

</front>

---

<back>

The `dbt deps` command with the `--add-package` flag.

Example:
```bash
dbt deps --add-package dbt-labs/codegen@1.0.0
```

</back>
