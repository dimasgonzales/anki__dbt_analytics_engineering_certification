<front>
What **dbt\_project.yml** field is used to specify a custom list of directories where analyses are located, and what is the default value?
</front>

---

<back>

**analysis-paths**. Without specifying this config, dbt will not compile any files as analyses. However, the dbt init command populates this value as ["analyses"]

</back>
