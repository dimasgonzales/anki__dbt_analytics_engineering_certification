<front>

How does the dbt Cloud CLI differ from dbt Core regarding where commands are executed?

</front>

---

<back>

Unlike dbt Core which runs locally on your machine, the dbt Cloud CLI executes commands against dbt's Cloud infrastructure. This architecture enables features like 'Automatic Deferral', allowing you to build only modified models while referencing upstream production artifacts without running them locally.

</back>
