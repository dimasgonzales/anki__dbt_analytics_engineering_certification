<front>

You are writing a CI/CD script and need to generate a clean list of all model names located in the `models/staging/stripe` directory. What command achieves this?

</front>

---

<back>

Use `dbt ls` with the selection flag (`-m` or `--select`) targeting the directory and the output flag set to `name`.

Command:
```bash
dbt ls -m models/staging/stripe --output name
```

</back>
