<front>

Which CLI flag allows you to prevent specific packages or models from running during a dbt execution?

</front>

---

<back>

The `--exclude` flag.

It removes the specified nodes from the list of nodes generated by the `--select` criteria.

Example:
```bash
dbt build --exclude package:dbt_project_evaluator
```

</back>
