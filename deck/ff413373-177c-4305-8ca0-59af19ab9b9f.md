<front>
How can you deserialize a json or yaml string into an appropriate Python object primitive (dict, list, etc.)?
</front>

---

<back>

```
fromjson(my_string, optional_default)
```
```
fromyaml(my_string, optional_default)
```  
The reverse operations are also available (`tojson`, `toyaml`)

</back>
