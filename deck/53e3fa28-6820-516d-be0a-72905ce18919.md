<front>

What is the syntax to define a reusable documentation block in a Markdown file and how do you reference it in a YAML file?

</front>

---

<back>

Define the block in a `.md` file using `{% docs block_name %} ... {% enddocs %}`. Reference it in a `.yml` file using the `{{ doc('block_name') }}` function.

Example:
```yaml
description: "{{ doc('block_name') }}"
```

</back>
