<front>

What restrictions does the dbt Cloud IDE place on the default protected branch (typically `main`)?

</front>

---

<back>

The dbt Cloud IDE prevents users from directly editing files, formatting code, linting, or executing dbt commands while on the protected branch. Users must create a feature branch to perform these actions.

</back>
