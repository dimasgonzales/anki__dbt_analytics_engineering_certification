<front>

Which dbt command produces the `sources.json` artifact instead of `run_results.json`?

</front>

---

<back>

The `dbt source freshness` command.

Unlike execution commands (like `run`, `test`, `build`) which generate `run_results.json` to track node execution, `source freshness` is a distinct operation that outputs its results specifically to `sources.json`.

</back>
