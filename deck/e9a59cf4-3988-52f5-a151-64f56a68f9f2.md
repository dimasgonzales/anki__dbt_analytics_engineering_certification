<front>

How do you restrict a dbt project to run only on specific versions of dbt Core to ensure consistency across the team?

</front>

---

<back>

You define the `require-dbt-version` configuration in your `dbt_project.yml` file. This setting accepts a version string or a range to enforce compatibility.

Example:
```yaml
require-dbt-version: ">=1.5.0 <1.6.0"
```

</back>
