<front>

What is the syntax format for a Fully Qualified Reference of a dimension in the dbt Semantic Layer?

</front>

---

<back>

The format uses the primary entity name followed by a double underscore and the dimension name:

```
entity_name__dimension_name
```

This ensures uniqueness across different semantic models.

</back>
