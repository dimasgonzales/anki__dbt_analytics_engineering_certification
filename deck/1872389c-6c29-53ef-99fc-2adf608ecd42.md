<front>

When running dbt in a Development environment, which environment of an upstream project does a cross-project `ref()` resolve to by default?

</front>

---

<back>

By default, it resolves to the upstream project's **Production** environment. This ensures developers build against the reliable source of truth unless an alternative resolution strategy (like Staging) is explicitly enabled.

</back>
