<front>

Which tab in the dbt Cloud Job Run interface allows you to download generated metadata files like `manifest.json` and `run_results.json`?

</front>

---

<back>

The **Artifacts** tab. This tab preserves the build artifacts for every specific job run (with history available for up to 365 days), allowing for metadata auditing and debugging of past runs.

</back>
