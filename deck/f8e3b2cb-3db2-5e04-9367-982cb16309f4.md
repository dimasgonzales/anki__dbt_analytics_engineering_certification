<front>

How can you optimize performance when running multiple dbt commands sequentially in a Python script to avoid repeated file parsing?

</front>

---

<back>

Parse the project once to generate a `Manifest` object, then pass this object to the constructor of subsequent `dbtRunner` instances.

```python
# Parse once
manifest = dbtRunner().invoke(["parse"]).result
# Reuse
dbt = dbtRunner(manifest=manifest)
```

</back>
