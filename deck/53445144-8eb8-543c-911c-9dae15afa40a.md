<front>

What CLI flag can you use to globally override project configurations and bypass SQL validation checks?

</front>

---

<back>

The `--static-analysis` flag. Setting this to `off` allows you to run commands without the parser validating the SQL syntax first.

```bash
dbt run --static-analysis off
```

</back>
