<front>

You need to run a dbt job that fails on any warning EXCEPT for Deprecation warnings. How would you write the CLI command?

</front>

---

<back>

You would use `--warn-error-options` to set 'error' to 'all', while explicitly defining 'Deprecations' in the 'warn' list.

Command:
```bash
dbt run --warn-error-options '{"error": "all", "warn": ["Deprecations"]}'
```

</back>
