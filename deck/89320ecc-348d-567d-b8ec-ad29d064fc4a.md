<front>

What does a `NULL` value in the `dbt_valid_to` column of a snapshot table indicate?

</front>

---

<back>

It indicates that the record is the current (active) version of that row. The `dbt_valid_to` column tracks the timestamp when a row version became invalid; if it is null, the row is still valid.

</back>
