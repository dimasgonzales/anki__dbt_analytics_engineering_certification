<front>

What is the primary responsibility of the Staging Layer regarding data types for sources like Google Sheets or flat files?

</front>

---

<back>

The Staging Layer is responsible for cleaning and casting raw data into the correct data types (e.g., casting string currency to numeric or timestamps to date types). This ensures downstream models can rely on consistent data types.

</back>
