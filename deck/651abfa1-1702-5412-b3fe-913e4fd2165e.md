<front>

Which file acts as the root of a dbt project and contains essential configurations like the project name and profile?

</front>

---

<back>

The `dbt_project.yml` file.

This file provides the context dbt needs to run, including project naming, versioning, and configurations for resources like models, seeds, and snapshots.

</back>
