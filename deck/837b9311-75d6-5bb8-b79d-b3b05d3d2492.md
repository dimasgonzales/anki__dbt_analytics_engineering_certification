<front>

After updating a package version in `packages.yml`, you run `dbt run` but receive a 'package not found' or compilation error. How do you resolve this?

</front>

---

<back>

You need to clear stale artifacts and install the new dependencies. Run the following commands:
```bash
dbt clean
dbt deps
```
This ensures the `dbt_packages/` directory is refreshed with the correct version specified in your configuration.

</back>
