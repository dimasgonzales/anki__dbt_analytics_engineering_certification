<front>

Your nightly dbt job has slowed down significantly. How would you use the Discovery API to pinpoint the specific models causing the delay?

</front>

---

<back>

You should query the `models` node and request the `executionInfo` object. By sorting models based on high `executionTime` values, you can identify which specific materializations are acting as bottlenecks.

```graphql
executionInfo {
  executionTime
}
```

</back>
