<front>

You have a model that should only apply a specific `WHERE` filter during incremental runs. How would you implement this in dbt?

</front>

---

<back>

Use the `is_incremental()` macro inside an `if` block to conditionally generate the SQL.

Example:
```sql
select * from source
{% if is_incremental() %}
  where event_time > (select max(event_time) from {{ this }})
{% endif %}
```

</back>
