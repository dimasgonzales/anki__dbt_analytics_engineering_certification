<front>

A project has a default selector defined to run only the `finance` package. If a user runs `dbt run --select tag:marketing`, what models will execute?

</front>

---

<back>

Only models tagged `marketing` will run.

When any CLI selection flag (`--select`, `--exclude`, etc.) is present, dbt ignores the `default: true` selector entirely; it does not merge the default selector with the CLI arguments.

</back>
