<front>

You need to instantly spin up a staging environment that mirrors production for a hotfix, but the dataset is petabytes in size. How do you accomplish this without a long data copy process?

</front>

---

<back>

Use `dbt clone` with the production state. If your data warehouse supports zero-copy cloning, dbt will create pointers to the production data almost instantly.

Command:
```bash
dbt clone --state path/to/prod/artifacts
```

</back>
