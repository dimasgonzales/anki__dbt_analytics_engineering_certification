<front>

How do you configure dbt to parse a seed file that uses a separator other than a comma (such as a pipe or semicolon)?

</front>

---

<back>

You must configure the `delimiter` property for the seed. This can be done in `dbt_project.yml` or inside a config block in a properties YAML file.

Example:
```yaml
seeds:
  +delimiter: "|"
```

</back>
