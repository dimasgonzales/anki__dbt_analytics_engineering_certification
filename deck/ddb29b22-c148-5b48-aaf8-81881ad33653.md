<front>

In which file must you define your database connection credentials (host, user, password) and target environments?

</front>

---

<back>

In the `profiles.yml` file.

This file is separate from the project logic (`dbt_project.yml`) to keep sensitive credentials out of version control. It typically resides in the `~/.dbt/` directory locally.

</back>
