<front>

Your Snowflake environment defaults to Python 3.8, but you need to use a library feature available only in Python 3.10 for a specific model. How do you configure this in dbt?

</front>

---

<back>

You should override the default platform version by setting the `python_version` configuration in the specific model.

```python
def model(dbt, session):
    dbt.config(
        materialized='table',
        python_version='3.10'
    )
    ...
```

</back>
