<front>

How do you reference an upstream model dependency within a dbt Python model?

</front>

---

<back>

You use the `dbt.ref()` method inside the Python `model` function. This works exactly like the Jinja `{{ ref() }}` macro in SQL, ensuring lineage and dependency ordering.

```python
# Returns a DataFrame referencing the upstream model
my_df = dbt.ref('upstream_model_name')
```

</back>
