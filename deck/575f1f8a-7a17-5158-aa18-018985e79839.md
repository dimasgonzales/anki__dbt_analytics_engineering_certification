<front>

A specific job requires more processing power than the project default. How would you increase the thread count for just that environment using Extended Attributes?

</front>

---

<back>

You can override the default settings by adding the `threads` key to the Extended Attributes block in the Environment configuration:

```yaml
threads: 8
```

</back>
