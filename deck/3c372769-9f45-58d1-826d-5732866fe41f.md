<front>

Your CI/CD pipeline fails immediately after running `dbt source freshness`. Assuming the command syntax was correct, what is the likely cause?

</front>

---

<back>

One or more sources exceeded their `error_after` threshold. When dbt detects data that is 'stale' beyond the error threshold, it returns a non-zero exit code, which typically causes pipelines to fail.

</back>
