<front>

You are trying to generate an embed code for a Data Health Tile, but the option is missing in the dbt Catalog. Your exposure is defined as `type: notebook`. How do you fix this?

</front>

---

<back>

You must update the exposure definition in your YAML file to use **`type: dashboard`**. The embeddable tile feature is exclusively available for exposures defined as dashboards.

</back>
