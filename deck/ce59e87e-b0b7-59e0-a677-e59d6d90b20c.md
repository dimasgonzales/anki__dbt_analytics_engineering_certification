<front>

Which YAML scalar style character is required to preserve line breaks exactly as written in a description, enabling complex Markdown formatting like lists or headers?

</front>

---

<back>

The **literal style** character (`|`).

It instructs the YAML parser to keep all newlines, which is necessary for Markdown elements like bullet points to render correctly by dbt.

</back>
