<front>

In dbt YAML configuration, what effect does the folded style indicator (>) have on multi-line text descriptions?

</front>

---

<back>

The folded style (`>`) replaces single newlines within the text with spaces. This allows you to write long descriptions across multiple lines in your code for readability while rendering them as a single continuous paragraph in the generated documentation.

</back>
