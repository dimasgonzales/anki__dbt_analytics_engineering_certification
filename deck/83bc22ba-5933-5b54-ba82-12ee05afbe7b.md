<front>

What configuration key must be set to instruct dbt to use BigQuery DataFrames (BigFrames) instead of Spark/Dataproc for a Python model?

</front>

---

<back>

You must set the `submission_method` configuration to `bigframes`. 

This tells dbt to transpile the pandas-like code into SQL for execution directly in BigQuery via the BigFrames API, rather than spinning up a Spark cluster.

</back>
