<front>

Which configuration parameters are required to materialize a model as a Snowflake Dynamic Table with clustering?

</front>

---

<back>

You must set `materialized='dynamic_table'`, define the `target_lag`, and use `cluster_by` for the clustering keys.
```sql
{{ config(
    materialized='dynamic_table',
    cluster_by=['user_id'],
    target_lag='1 hour'
) }}
```

</back>
