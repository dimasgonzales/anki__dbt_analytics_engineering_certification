<front>

In BigQuery `insert_overwrite` models, what variable represents the maximum existing partition in the destination table?

</front>

---

<back>

The variable is `_dbt_max_partition`. It is a BigQuery scripting variable automatically generated by dbt during the incremental run to help filter source data for only new partitions.

</back>
