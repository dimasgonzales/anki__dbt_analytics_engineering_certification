<front>

What syntax is used to define a documentation block in a Markdown file, and how do you reference it in a YAML file?

</front>

---

<back>

**Definition (.md):**
```jinja
{% docs block_name %}
Content...
{% enddocs %}
```

**Reference (.yml):**
```yaml
description: '{{ doc("block_name") }}'
```

</back>
