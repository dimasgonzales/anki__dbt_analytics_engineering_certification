<front>

How do you configure a default access level for all models within a specific directory using `dbt_project.yml`?

</front>

---

<back>

Use the `+access` configuration key nested under the specific directory hierarchy in your `dbt_project.yml` file.

Example:
```yaml
models:
  my_project:
    staging:
      +access: private
```
This ensures all models in the staging folder are private by default.

</back>
