<front>

What is the primary purpose of the **Compare** tab available in dbt Cloud CI jobs (Enterprise)?

</front>

---

<back>

To visualize the differences between the production environment and the code in a pull request. It highlights schema changes (such as added or removed columns) and modified rows to validate the impact of logic changes before merging.

</back>
