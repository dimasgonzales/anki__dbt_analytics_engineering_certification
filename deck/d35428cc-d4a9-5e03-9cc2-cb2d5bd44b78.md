<front>

You need to programmatically audit the success/failure status of every model in a specific historical job run. Which file should you retrieve and where is it located?

</front>

---

<back>

You should retrieve the `run_results.json` file. It is located in the **Artifacts** tab of the specific Job Run in dbt Cloud. This file contains the detailed status, timing, and results for every node executed in that run.

</back>
