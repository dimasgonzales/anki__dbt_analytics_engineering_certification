<front>

Which Jinja control structure is used to iterate over a list of values to dynamically generate repetitive SQL code (such as pivoting columns)?

</front>

---

<back>

The `{% for item in list %} ... {% endfor %}` loop is used. It repeats the code block inside the loop for every item in the provided list context.

</back>
