<front>

You need to limit the data processed in your `orders` model to the last 3 days only when running in the 'dev' environment to speed up iteration. How do you accomplish this?

</front>

---

<back>

Add a Jinja conditional block in the `where` clause of your model checking `target.name`.

```sql
select * from {{ source('app', 'orders') }}
{% if target.name == 'dev' %}
  where order_date > dateadd('day', -3, current_date)
{% endif %}
```

</back>
