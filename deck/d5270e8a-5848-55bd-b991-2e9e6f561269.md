<front>

You are setting up a CI pipeline and want to cheaply validate that a Pull Request does not contain SQL syntax errors. Which command should you use?

</front>

---

<back>

```bash
$ dbt build --empty
```

This command builds the lineage and validates the code against the warehouse schema but filters queries to return 0 rows, keeping costs minimal.

</back>
