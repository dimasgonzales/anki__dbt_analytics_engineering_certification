<front>

How do you reference a seed within a dbt model to ensure the dependency is tracked in the DAG?

</front>

---

<back>

Use the `ref()` function, treating the seed exactly like a model:

```sql
select * 
from {{ ref('seed_file_name') }}
```

</back>
