<front>

What is the specific selection syntax used with the `--select` flag to run freshness checks only for tables within a specific source named `marketing`?

</front>

---

<back>

You use the `source:` prefix followed by the source name.

Command:
```bash
dbt source freshness --select source:marketing
```

</back>
