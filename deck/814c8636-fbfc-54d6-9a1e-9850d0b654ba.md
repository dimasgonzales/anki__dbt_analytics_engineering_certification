<front>

What specific syntax must be used to select data from a cross-project model in dbt?

</front>

---

<back>

You must use the two-argument `ref` function specifying both the project name and model name.

Example:
```sql
select * from {{ ref('project_name', 'model_name') }}
```

</back>
