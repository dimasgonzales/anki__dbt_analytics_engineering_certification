<front>

How do you force dbt to ignore the `package-lock.yml` file and fetch the latest compatible versions of dependencies?

</front>

---

<back>

Run the deps command with the upgrade flag:
```bash
dbt deps --upgrade
```
This updates the lock file with the newest versions that match the constraints defined in `packages.yml`.

</back>
