<front>

What is the *only* valid location to configure flags that opt-in or opt-out of specific dbt behavior changes (such as legacy migrations)?

</front>

---

<back>

The **`flags` dictionary** in **`dbt_project.yml`**.

```yaml
# dbt_project.yml
flags:
  some_legacy_behavior: true
```
These cannot be set via CLI or Environment Variables.

</back>
