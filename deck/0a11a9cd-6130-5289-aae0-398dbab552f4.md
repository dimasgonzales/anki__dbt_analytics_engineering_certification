<front>

You have a Python library installed on your local laptop, but your dbt Python model fails with a "Module Not Found" error during execution. What is the likely cause?

</front>

---

<back>

Since dbt Python models run remotely in the cloud data warehouse (not locally), the environment where the code executes must have the library installed. Your local laptop's Python environment does not affect the execution environment of the data warehouse.

</back>
