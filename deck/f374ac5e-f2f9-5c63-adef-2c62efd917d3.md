<front>

How do you reference a reusable documentation block defined in a Markdown file within a YAML property?

</front>

---

<back>

Use the `doc()` Jinja function within the YAML file, passing the name of the documentation block as the argument.

```yaml
description: "{{ doc('my_reusable_doc_block') }}"
```

</back>
