<front>

What is the difference between the `set` and `set_strict` functions?

</front>

---

<back>

`set` allows for an optional "default" argument that determines what the function returns if the given value cannot be converted to a set.
  

```
set_strict
``` will throw a TypeError if passed a non-iterable value.

</back>
