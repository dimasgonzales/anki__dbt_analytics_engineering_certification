<front>

How do you modify the default configuration (e.g., timezone variables) of an installed dbt package without modifying the package source code?

</front>

---

<back>

Define variables in your root `dbt_project.yml` file under the `vars` key.

Example:
```yaml
vars:
  package_name:
    variable_name: value
```

</back>
