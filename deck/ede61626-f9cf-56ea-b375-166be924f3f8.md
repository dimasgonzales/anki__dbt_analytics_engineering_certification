<front>

You need to ensure that the raw data loaded into your warehouse is up-to-date before running your models. How does defining Sources help with this?

</front>

---

<back>

Defining Sources allows you to configure **freshness rules** (via `freshness:` blocks). You can then run:
```bash
dbt source freshness
```
This checks if the data arrival times meet your criteria before you trigger the transformation pipeline.

</back>
