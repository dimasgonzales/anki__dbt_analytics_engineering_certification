<front>

Can you use Jinja logic within dbt YAML configuration files (like `sources.yml`)?

</front>

---

<back>

Yes, dbt renders `.yml` files using Jinja. This allows for dynamic configuration, such as setting database names, schemas, or tables based on environment variables or the `target` object.

</back>
