<front>

Your database contains a table named `TRN_ORD_2023_X`. You want to reference this table simply as `orders` in your dbt models using `{{ source('retail', 'orders') }}`. How should you configure the source YAML?

</front>

---

<back>

Use the `identifier` property to map the logical name to the physical name.

```yaml
sources:
  - name: retail
    tables:
      - name: orders
        identifier: TRN_ORD_2023_X
```

</back>
