<front>

You need to rebuild a specific model, `fct_revenue`, and ensure that any models relying on it are also updated to reflect the changes. Which command would you use?

</front>

---

<back>

```bash
dbt run --select fct_revenue+
```
Placing the `+` operator *after* the model name selects the model itself and all downstream dependencies.

</back>
