<front>

In dbt Core, how do you execute a command against a specific environment configuration defined in `profiles.yml`?

</front>

---

<back>

Use the `--target` flag followed by the name of the output defined in your profile. For example, to run against the 'prod' output:
```bash
dbt run --target prod
```

</back>
