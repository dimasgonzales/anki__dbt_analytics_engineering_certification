<front>

What Jinja function is used in YAML descriptions to reference a documentation block defined in a separate Markdown file?

</front>

---

<back>

The `doc()` function. It inserts content defined within `{% docs block_name %}` tags from `.md` files.

Usage:
```yaml
description: "{{ doc('my_doc_block') }}"
```

</back>
