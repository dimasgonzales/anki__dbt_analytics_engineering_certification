<front>

What is the purpose of the `config-version` key in `dbt_project.yml`?

</front>

---

<back>

The `config-version` key tells dbt how to interpret the structure of the configuration file. It ensures clarity and forward compatibility by explicitly defining which syntax version the project uses (currently version 2).

</back>
