<front>

You navigate into your `models/` subdirectory and run `dbt run`. You receive a fatal error stating `dbt_project.yml` could not be found. How do you troubleshoot this?

</front>

---

<back>

Change your directory context to the **project root**. dbt commands generally need to be run from the root folder containing the `dbt_project.yml` file.

```bash
cd ..
dbt run
```

</back>
