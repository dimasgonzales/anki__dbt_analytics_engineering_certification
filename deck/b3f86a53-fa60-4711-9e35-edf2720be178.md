How can you access a selected group of useful Python modules for use in your Jinja code, and what modules are available?
</front>

---

<back>

The `modules` variable is a pointer/namespace containing the following Python modules:
  

- datetime
- pytz
- re
- itertools

Example:

`{% set now = modules.datetime.datetime.now() %}`
