<front>

Which three Jinja tags are primarily used to construct a conditional control flow structure in dbt?

</front>

---

<back>

The structure relies on:
1. `{% if condition %}` to start the logic.
2. `{% else %}` (optional) for alternative logic.
3. `{% endif %}` to strictly close the block.

Failure to close with `{% endif %}` will cause a compilation error.

</back>
