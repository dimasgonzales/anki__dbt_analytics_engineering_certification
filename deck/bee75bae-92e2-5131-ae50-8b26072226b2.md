<front>

You are debugging a complex macro and want to see the value of a specific variable in your terminal during a run. How can you accomplish this?

</front>

---

<back>

Use the `log` function within a Jinja expression and set the `info` argument to `True`.

Example:
```sql
{{ log(my_variable, info=True) }}
```

</back>
