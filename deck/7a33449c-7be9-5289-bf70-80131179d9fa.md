<front>

How is the syntax for checking incremental state different in a Python model compared to a SQL model?

</front>

---

<back>

In SQL models, you use the Jinja macro `{{ is_incremental() }}`. In Python models, you access the boolean attribute `dbt.is_incremental` on the `dbt` object.

```python
if dbt.is_incremental:
    # logic
```

</back>
