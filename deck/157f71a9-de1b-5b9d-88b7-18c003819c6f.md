<front>

Which specific file must be present in the current working directory for the dbt CLI to successfully establish a project context?

</front>

---

<back>

The `dbt_project.yml` file. The dbt CLI looks for this file in the current directory to define the project context; without it, dbt cannot determine project paths or configurations.

</back>
