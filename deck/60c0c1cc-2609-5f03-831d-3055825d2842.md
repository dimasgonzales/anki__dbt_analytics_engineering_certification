<front>

How can you execute the `dbt seed` command for a single specific CSV file?

</front>

---

<back>

You can use the `--select` (or `-s`) flag followed by the name of the seed (without the .csv extension).

```bash
dbt seed --select specific_seed_name
```

</back>
