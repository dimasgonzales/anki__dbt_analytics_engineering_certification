<front>

You need to embed a dbt Data Health Tile into a public Notion page. How should you handle authentication for the iframe?

</front>

---

<back>

Create a **Service Token** with **Metadata Only** permissions. Do not use a personal developer token or an Account Admin token, as this would expose sensitive write-access capabilities in a potentially insecure environment.

</back>
