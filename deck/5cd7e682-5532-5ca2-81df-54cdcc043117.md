<front>

Which command is used to verify that raw source data has been loaded within a specific time threshold defined in the project configuration?

</front>

---

<back>

```bash
dbt source freshness
```

This calculates the recency of records in source tables against the `freshness` block defined in your source YAML files.

</back>
