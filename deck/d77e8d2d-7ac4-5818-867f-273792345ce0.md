<front>

You have just cloned a new dbt repository that utilizes several external libraries (such as `dbt-utils`) defined in `packages.yml`. You immediately try to run `dbt run` but receive an error that the packages cannot be found. What must you do to resolve this?

</front>

---

<back>

You must run the following command to install the required packages into your project:

```bash
dbt deps
```

</back>
