<front>

You have a macro `get_status_codes()` that contains `{% do return([200, 400, 500]) %}`. 

In a model, you run `{% set codes = get_status_codes() %}`. If you check `{{ codes is string }}`, will it be true or false?

</front>

---

<back>

It will be **false**. 

Because the macro uses `return` inside a `do` block, the variable `codes` is set to an actual Python **list**, not a string representation of that list.

</back>
