<front>

What does the `dbt compile` command do?

</front>

---

<back>

It generates executable SQL from source model, test, and analysis files. The compiled files are placed in your configured target directory. 

"The `compile` command is useful for: 
1. Visually inspecting the compiled output of model files. 
2. Manually running compiled SQL. 
3. Compiling `analysis` files."

</back>
