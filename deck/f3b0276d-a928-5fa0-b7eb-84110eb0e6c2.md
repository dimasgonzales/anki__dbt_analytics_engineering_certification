You have a model `fct_orders` with versions `v1`, `v2`, and `v3`. You want to run only version 2. What CLI command should you use?

</front>

---

<back>

Use dot notation to select the specific version.

Command:
```bash
dbt run --select fct_orders.v2
```

</back>
