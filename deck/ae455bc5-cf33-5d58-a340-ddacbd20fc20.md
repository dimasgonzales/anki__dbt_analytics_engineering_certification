<front>

Your CI pipeline is taking too long because it rebuilds the entire warehouse on every commit. How can you configure dbt to only build changed models and their downstream children?

</front>

---

<back>

Use the `state:modified` selector with the `+` graph operator and provide the path to production artifacts.

Command:
```bash
dbt run --select state:modified+ --state path/to/prod_artifacts
```

</back>
