<front>

How do you access complex data structures (like lists or dictionaries) defined in `dbt_project.yml` within a model or macro?

</front>

---

<back>

Use the `var()` function.

dbt parses the YAML configuration into native Python objects, so `var('my_list_variable')` returns a list you can iterate over using Jinja.

</back>
