<front>

How can you filter dbt resources based on properties defined in their configuration block, such as materialization strategy?

</front>

---

<back>

Use the `config:` selector followed by the property name and value. This works for standard configs and custom meta fields.

```bash
# Example: Run only incremental models
dbt run --select "config.materialized:incremental"
```

</back>
