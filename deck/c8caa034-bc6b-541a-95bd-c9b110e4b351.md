<front>

You have defined a documentation block named `order_status_codes`. How would you apply this description to the `status` column in your `models.yml` file?

</front>

---

<back>

Use the `doc()` function inside the Jinja brackets within the `description` field of the column configuration.

Code:
```yaml
columns:
  - name: status
    description: "{{ doc('order_status_codes') }}"
```

</back>
