<front>

How do you configure a dbt selector to run only resources that satisfy multiple criteria simultaneously (Intersection logic)?

</front>

---

<back>

Separate the selector arguments with a **comma (`,`)** and **no whitespace**. This tells dbt to select only resources that match *all* provided arguments.

Example:
```bash
dbt run --select path/to/models,tag:hourly
```

</back>
