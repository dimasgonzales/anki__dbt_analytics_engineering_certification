<front>

What configuration properties are used to define conditional failure thresholds for dbt tests?

</front>

---

<back>

You can use `warn_if` and `error_if` configurations. These accept conditional logic (e.g., `>10`) to determine when a test result should trigger a warning or an error based on the number of failing rows returned.

</back>
