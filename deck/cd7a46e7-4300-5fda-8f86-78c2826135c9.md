<front>

A developer has edited a model and verified it runs successfully in their development schema. What are the next steps in the git workflow to get this code into production?

</front>

---

<back>

1. **Commit** the changes to the feature branch.
2. Open a **Pull Request (PR)** to merge into the main branch.

This initiates code review and CI testing before deployment.

</back>
