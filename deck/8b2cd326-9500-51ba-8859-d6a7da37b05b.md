<front>

What specific error does dbt raise during parsing when a model attempts to reference a private model from a different group?

</front>

---

<back>

dbt raises a `DbtReferenceError`. This stops the run immediately during the parsing/compilation phase because the dependency violates the configured access governance.

</back>
