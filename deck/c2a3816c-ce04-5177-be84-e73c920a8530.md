<front>

You applied the tag `daily_update` to several models and seeds. When you execute `dbt run --select tag:daily_update`, the seeds do not rebuild. Why?

</front>

---

<back>

The `dbt run` command is specific to building models (views/tables). It filters out other resource types. To rebuild the seeds with that tag, you must use:

```bash
dbt seed --select tag:daily_update
```

</back>
