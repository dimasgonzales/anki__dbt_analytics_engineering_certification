<front>

You are using a generic 'Git clone' connection for a Bitbucket repository. How would you automate a CI job to run when a Pull Request is opened?

</front>

---

<back>

You must use the **dbt Cloud Administrative API**. Because generic git connections lack native CI/CD integration, you must configure an external webhook in Bitbucket to trigger the job via the API.

</back>
