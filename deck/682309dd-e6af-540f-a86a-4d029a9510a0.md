<front>

Which two selectors allow you to distinguish between 'unit' vs 'data' tests, and specific generic test definitions (like 'unique')?

</front>

---

<back>

1. `test_type:` filters by category (`unit` or `data`).
2. `test_name:` filters by the specific name of the generic test definition.

```bash
dbt test --select "test_type:unit"
dbt test --select "test_name:unique"
```

</back>
