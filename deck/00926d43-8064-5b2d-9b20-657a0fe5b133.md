<front>

What three specific conditions must be met for the `is_incremental()` macro to return `True`?

</front>

---

<back>

1. The destination table already exists in the database.
2. The run is NOT being executed with the `--full-refresh` flag.
3. The model configuration is set to `materialized='incremental'`.

</back>
