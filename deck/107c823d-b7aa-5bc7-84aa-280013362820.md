<front>

You have just installed dbt on a new machine and want to verify that your project configuration is syntactically correct and your database connection is working. What is the first command you should run?

</front>

---

<back>

You should run:
```bash
dbt debug
```
This serves as a health check to diagnose installation issues or syntax errors before attempting to compile or run models.

</back>
