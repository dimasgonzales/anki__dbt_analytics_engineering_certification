<front>

You are troubleshooting a 'parsing error' on a model you believe contains valid SQL. How can you quickly verify if the error is a false positive without editing config files?

</front>

---

<back>

Run the specific model with the static analysis flag set to off. If it runs successfully, the SQL is valid and the parser was incorrect.

```bash
dbt run --select my_model --static-analysis off
```

</back>
