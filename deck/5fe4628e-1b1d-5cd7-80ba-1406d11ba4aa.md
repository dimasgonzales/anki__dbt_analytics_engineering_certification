<front>

What is the specific difference between `dbt compile` and `dbt run`?

</front>

---

<back>

**dbt compile** resolves Jinja/YAML into raw SQL and saves it to the `target/` folder without executing it.

**dbt run** compiles the code AND executes the resulting DDL/DML against the data warehouse to create objects.

</back>
