<front>

What specific behavior does the `--connection` flag trigger when running `dbt debug`?

</front>

---

<back>

The `--connection` flag forces `dbt debug` to skip project parsing, dependency checks, and file validation. It attempts *only* to authenticate with the data warehouse.

</back>
