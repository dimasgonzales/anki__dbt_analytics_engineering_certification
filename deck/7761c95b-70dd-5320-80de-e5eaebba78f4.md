<front>

What CLI syntax pattern allows you to run all models that are downstream dependencies of a specific source?

</front>

---

<back>

Use the `source:` selector followed by the source name and the graph operator `+`.

```bash
$ dbt run --select source:my_source_name+
```

</back>
