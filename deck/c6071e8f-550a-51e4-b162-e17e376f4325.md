<front>

What does the context variable `this` return in a dbt model?

</front>

---

<back>

The `this` variable returns the relation object for the current model being executed. It includes attributes for the `database`, `schema`, and `identifier` (table/view name).

It is commonly used in incremental models or hooks to reference the destination table.

</back>
