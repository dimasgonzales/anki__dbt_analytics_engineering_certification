<front>

What dbt CLI command allows you to execute a macro as a standalone task, separate from the DAG execution?

</front>

---

<back>

The `dbt run-operation` command.

It is designed for maintenance tasks, administrative operations, or utility scripts (like granting permissions) that do not generate database objects as part of the standard model build process.

</back>
