<front>

What is the primary function of the `dbt_utils.generate_surrogate_key` macro, and what algorithm does it use by default?

</front>

---

<back>

It generates a deterministic, unique surrogate key by hashing a list of input columns. By default, it uses the MD5 cryptographic hash function to ensure the same inputs always produce the same output string.

</back>
