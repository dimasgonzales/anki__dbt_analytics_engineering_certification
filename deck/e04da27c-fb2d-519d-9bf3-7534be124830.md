<front>

What three hierarchical attributes does dbt use to uniquely identify a specific source table?

</front>

---

<back>

Dbt identifies sources using **Database**, **Schema**, and **Identifier** (table name). When using the `{{ source() }}` function or configuring `sources.yml`, this combination ensures the specific object is correctly located in the warehouse.

</back>
