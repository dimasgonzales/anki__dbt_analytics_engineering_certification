<front>

You need to apply a specific Row-Level Security (RLS) policy to a table created by dbt. How should you implement this?

</front>

---

<back>

Use a `post-hook`, as standard grants does not support RLS.
```sql
{{ config(
  post_hook = "alter table {{ this }} add row access policy ..."
) }}
```

</back>
