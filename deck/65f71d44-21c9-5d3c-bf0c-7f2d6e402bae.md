<front>

You need to add a metadata tag `sensitive` to the `email` column in the `users` model for downstream governance tools. How do you apply this in YAML?

</front>

---

<back>

Use a `config` block nested strictly under the specific column.

```yaml
columns:
  - name: email
    config:
      tags: ['sensitive']
```

</back>
