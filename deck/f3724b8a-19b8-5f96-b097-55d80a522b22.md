<front>

A marketing dashboard requires data to be updated every hour, but the full warehouse refresh takes 3 hours. How can you solve this using dbt jobs?

</front>

---

<back>

Create a dedicated job scheduled to run hourly that targets only the dependencies of that specific dashboard. Configure the job command to use the selection syntax `+exposure:marketing_dashboard` to limit the run to only the models required by that exposure.

</back>
