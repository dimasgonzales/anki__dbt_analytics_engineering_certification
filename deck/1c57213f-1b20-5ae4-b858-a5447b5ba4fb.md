<front>

A macro fails with a `TypeError` when trying to log a message using `{{ print('Rows: ' + row_count) }}` where `row_count` is an integer. How do you fix this?

</front>

---

<back>

Replace the plus (`+`) operator with the tilde (`~`) operator. The `+` operator strictly requires matching types (and tries to add numbers), while `~` converts all inputs to strings before joining them.

Corrected:
```jinja
{{ print('Rows: ' ~ row_count) }}
```

</back>
