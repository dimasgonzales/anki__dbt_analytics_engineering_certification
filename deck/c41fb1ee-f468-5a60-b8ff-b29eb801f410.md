<front>

You need to ensure the `payment_method` column in your `stg_payments` model only contains 'credit_card', 'coupon', or 'bank_transfer'. How do you configure this test in YAML?

</front>

---

<back>

Use the `accepted_values` test with a list of valid options:

```yaml
columns:
  - name: payment_method
    data_tests:
      - accepted_values:
          values: ['credit_card', 'coupon', 'bank_transfer']
```

</back>
