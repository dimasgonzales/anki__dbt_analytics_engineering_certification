<front>

Which configuration key in `dbt_project.yml` is used to specify custom directories for seed files?

</front>

---

<back>

The `seed-paths` configuration key. It accepts a list of directory paths relative to the project root where dbt should look for `.csv` files to load as seeds.

```yaml
seed-paths: ["seeds", "other_data"]
```

</back>
