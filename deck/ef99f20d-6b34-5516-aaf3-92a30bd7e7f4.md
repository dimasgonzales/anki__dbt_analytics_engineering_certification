<front>

What specific data format does the `selected_resources` variable contain?

</front>

---

<back>

It contains a list of strings representing the fully qualified **Node IDs** of the resources matching the CLI selection criteria. An example value in the list would be `'model.project_name.model_name'`.

</back>
