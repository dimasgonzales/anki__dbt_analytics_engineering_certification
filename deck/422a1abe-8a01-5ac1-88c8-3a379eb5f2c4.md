<front>

How does the `--select` flag behave differently when used with `dbt docs generate` compared to `dbt run`?

</front>

---

<back>

In `dbt docs generate`, the `--select` flag limits ONLY the database metadata queries (catalog generation). It does **not** filter the compilation step; dbt still parses and compiles the entire project into `manifest.json`, ensuring the full lineage graph remains available.

</back>
