<front>

You want to deploy a staging model `stg_payments` but hide it from the auto-generated dbt Documentation site to reduce clutter. How do you configure this?

</front>

---

<back>

Configure the `docs` key inside the model's config block setting `show` to false.

```yaml
models:
  - name: stg_payments
    config:
      docs:
        show: false
```

</back>
