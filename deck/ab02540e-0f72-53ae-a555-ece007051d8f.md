<front>

What command is used to display the installed version of dbt Core and list all installed adapter plugins?

</front>

---

<back>

The command `dbt --version` is used. It outputs the specific version of dbt Core installed locally and lists all installed plugins (adapters) like Snowflake or BigQuery along with their versions.

Example output:
```text
Core:
  - installed: 1.7.6
Plugins:
  - snowflake: 1.7.1
```

</back>
