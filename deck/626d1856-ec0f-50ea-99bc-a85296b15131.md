<front>

How do you configure dbt to use different log verbosity levels for the CLI console versus the persistent log file?

</front>

---

<back>

Use `--log-level` for console output and `--log-level-file` for the log file.

Example:
```bash
# Info for console, Debug for file
dbt run --log-level info --log-level-file debug
```

</back>
