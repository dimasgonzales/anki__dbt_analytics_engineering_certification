<front>

Given the following source configuration, what will the function `{{ source('stripe', 'payments') }}` compile to in the generated SQL?

```yaml
sources:
  - name: stripe
    tables:
      - name: payments
        identifier: raw_stripe_charges
```

</front>

---

<back>

It will compile to `schema.raw_stripe_charges`.

Because an `identifier` is provided, dbt uses it for the physical table reference instead of the logical `name` (`payments`).

</back>
