<front>

You are reviewing a `packages.yml` file and see the following configuration that is causing an error. What is the problem?

```yaml
- package: dbt-labs/dbt_utils
  revision: 1.0.0
```

</front>

---

<back>

The syntax mixes Hub and Git concepts. Hub packages defined with the `package` key must use `version` (supporting semantic versioning). The `revision` key is reserved for packages defined with the `git` key.

</back>
