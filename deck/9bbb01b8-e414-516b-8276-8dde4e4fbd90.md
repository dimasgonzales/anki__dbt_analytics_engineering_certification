<front>

You need to run data tests specifically on all models configured as `incremental` to validate keys before a deployment. How would you construct the selector?

</front>

---

<back>

You can select based on the materialization configuration:

```bash
dbt test --select "config.materialized:incremental"
```
This targets all models with the specific configuration and runs their associated tests.

</back>
