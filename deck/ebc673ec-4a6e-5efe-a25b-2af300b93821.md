<front>

You need to add a private Git package to your project via the CLI without opening a text editor. How would you construct this command?

</front>

---

<back>

Use `--add-package` with the repository URL and specify the `--source git` flag.

Example:
```bash
dbt deps --add-package https://github.com/org/repo@v1.0 --source git
```

</back>
