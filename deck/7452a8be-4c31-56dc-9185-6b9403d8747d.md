<front>

How do you add a description to a specific generic test (like `unique` or `not_null`) in `schema.yml`?

</front>

---

<back>

Nest the `description` key immediately under the specific test assertion key within the column's `data_tests` list.

```yaml
columns:
  - name: id
    data_tests:
      - unique:
          description: "ID must be unique per row"
```

</back>
