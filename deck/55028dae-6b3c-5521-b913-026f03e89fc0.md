<front>

What Jinja syntax is used to define a documentation block in a dbt markdown file?

</front>

---

<back>

You must use the opening `{% docs block_name %}` tag and the closing `{% enddocs %}` tag. The name defined in the opening tag is used to reference the block later.

Example:
```markdown
{% docs table_description %}
This table contains...
{% enddocs %}
```

</back>
