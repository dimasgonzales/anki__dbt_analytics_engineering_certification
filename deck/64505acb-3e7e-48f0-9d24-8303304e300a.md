<front>
What **dbt\_project.yml** field is used to specify a custom directory where compiled files will be written when you run `dbt run`, `dbt compile`, or `dbt test` commands? What is the default? How else can this be configured?
</front>

---

<back>

**target-path**. Default is `target`. This can be configured in three ways (in descending order of precedence):  

1. "--target-path" CLI flag
2. "DBT\_TARGET\_PATH" environment variable
3. "target-path" field in dbt\_project.yml

</back>
