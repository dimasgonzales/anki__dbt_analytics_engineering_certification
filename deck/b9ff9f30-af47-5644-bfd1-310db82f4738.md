<front>

What is the primary difference between the data stored in `target/manifest.json` vs. `target/catalog.json` after running `dbt docs generate`?

</front>

---

<back>

`manifest.json` contains project resources, configuration, logic, and lineage (dbt-side information). `catalog.json` contains database metadata produced by querying the warehouse, such as column types, table statistics, and row counts (database-side information).

</back>
