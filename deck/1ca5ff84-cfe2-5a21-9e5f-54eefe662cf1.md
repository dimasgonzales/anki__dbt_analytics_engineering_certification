What are the two required arguments for the `model` function in a dbt Python model, and what is the mandatory return value?

</front>

---

<back>

The function must accept `dbt` and `session` as arguments and must return a **DataFrame**.

Example:
```python
def model(dbt, session):
    # logic here
    return df
```

</back>
