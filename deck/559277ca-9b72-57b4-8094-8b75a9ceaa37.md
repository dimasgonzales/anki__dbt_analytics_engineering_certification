<front>

How does dbt associate the data passed in the `--args` CLI flag with the variables defined in a macro?

</front>

---

<back>

dbt maps the **keys** in the YAML dictionary supplied to `--args` to the **argument names** in the macro's definition. The names must match exactly for the values to be passed correctly.

</back>
