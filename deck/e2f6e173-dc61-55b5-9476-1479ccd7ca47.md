<front>

You are writing a macro for a Snowflake project and need to access the current virtual warehouse name being used. How do you reference this?

</front>

---

<back>

You access it via the `target` object using the specific key defined in your `profiles.yml`. Note that this is an adapter-specific attribute.

```jinja
{{ target.warehouse }}
```

</back>
