<front>

What happens if you change the `snapshot_meta_column_names` configuration for a snapshot that has already been created in the database?

</front>

---

<back>

dbt will fail during the next run. dbt does not automatically rename existing columns in the database to match the new configuration. It will attempt to query using the new names, which do not exist in the physical table yet.

</back>
