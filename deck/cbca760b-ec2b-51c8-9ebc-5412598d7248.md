<front>

How do you select all models within a package or directory?

</front>

---

<back>

The **\*** operator, e.g.:   
```
$ dbt run --select snowplow.\* # run all of the models in the snowplow package
$ dbt run --select finance.base.\* # run all of the models in models/finance/base
```

</back>
