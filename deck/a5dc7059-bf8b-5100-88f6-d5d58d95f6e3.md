<front>

In a dbt macro, what function must be used to pass a data structure (like a list) back to the calling context rather than rendering it as a string?

</front>

---

<back>

The `return()` function.

Using `{{ return(my_variable) }}` passes the Python object back to the caller. Standard Jinja variable printing (e.g., `{{ my_variable }}`) converts the object to a string and renders it directly into the compiled SQL.

</back>
