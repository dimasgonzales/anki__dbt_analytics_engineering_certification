You are defining a dbt model using Python. Does the data need to be extracted from your data warehouse to your local machine or dbt Cloud to be processed?

---

<back>

**No.** dbt Python models execute within the data warehouse (or analytical engine like Databricks/Snowflake).

dbt sends the Python code to the warehouse to be executed natively, ensuring data does not move externally during transformation.

</back>
