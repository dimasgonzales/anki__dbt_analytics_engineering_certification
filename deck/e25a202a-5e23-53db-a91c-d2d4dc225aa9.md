<front>

What is a dbt seed and what command is used to load it into the warehouse?

</front>

---

<back>

A seed is a CSV file located in the `seeds/` directory of a dbt project, intended for static data like country codes or mapping tables. It is loaded into the data warehouse using the command:

```bash
dbt seed
```

</back>
