<front>

How do you declare Python library dependencies (such as scikit-learn or pandas) specific to a single dbt model?

</front>

---

<back>

You define dependencies using the `packages` argument within the `dbt.config()` method inside the model function. This ensures the specific environment requires these libraries for execution.

Example:
```python
dbt.config(
    packages=["scikit-learn==1.0.2", "pandas"]
)
```

</back>
