<front>

What dbt CLI command is used to validate that your `dbt_project.yml` and `profiles.yml` files are correctly configured and that a database connection can be established?

</front>

---

<back>

```bash
dbt debug
```
This command checks for the existence of required configuration files, validates the profile connection details, and attempts a test connection to the warehouse.

</back>
