<front>

You need to rebuild the model `fct_orders` and only its immediate parents (1 level upstream) for testing. What selection syntax should you use?

</front>

---

<back>

Use the plus sign (`+`) before the model name prefixed with the number `1`.

Command:
```bash
dbt run --select 1+fct_orders
```

</back>
