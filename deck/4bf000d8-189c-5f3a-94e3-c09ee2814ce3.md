<front>

How does dbt Cloud determine which dbt version to use for a specific Job by default?

</front>

---

<back>

By default, a Job **inherits** the dbt version and configuration from the **Environment** it is assigned to. 

However, this can be manually overridden at the Job level if a specific job requires a different version than the Environment.

</back>
