<front>

How can you test uniqueness across multiple columns using the standard `unique` test without using external packages?

</front>

---

<back>

Create a surrogate key in the SQL model by concatenating the relevant columns, then apply the `unique` test to this new column in the YAML configuration.

```sql
-- models/orders.sql
select country_code || '-' || order_id as surrogate_key, ...
```
```yaml
# models/orders.yml
columns:
  - name: surrogate_key
    data_tests:
      - unique
```

</back>
