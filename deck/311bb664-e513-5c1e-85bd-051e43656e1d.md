<front>

When debugging, what is the specific difference between the files found in `target/compiled/` versus `target/run/`?

</front>

---

<back>

- `target/compiled/`: Contains the pure `SELECT` statement with all Jinja compiled into SQL.
- `target/run/`: Contains the full DDL (Data Definition Language) statement actually executed by the database (e.g., wrapped in `create view ... as`).

</back>
