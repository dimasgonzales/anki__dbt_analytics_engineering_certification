<front>

Which dbt command allows you to generate the raw SQL for a model without executing it against the database, and where is the output stored?

</front>

---

<back>

The command is `dbt compile`.

The resolved SQL (with all Jinja logic executed and variables replaced) is stored in the `target/compiled/{project_name}/` directory.

</back>
