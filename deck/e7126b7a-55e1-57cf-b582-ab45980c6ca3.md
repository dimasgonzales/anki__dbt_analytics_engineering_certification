<front>

You need a dbt job to run daily at 5:00 PM EST (Eastern Standard Time). EST is UTC-5. What time/cron schedule should you configure in dbt Cloud?

</front>

---

<back>

You should configure the job to run at **22:00 UTC** (10:00 PM UTC).

Calculation: 17:00 (5 PM) + 5 hours = 22:00.

Cron syntax:
```text
0 22 * * *
```

</back>
