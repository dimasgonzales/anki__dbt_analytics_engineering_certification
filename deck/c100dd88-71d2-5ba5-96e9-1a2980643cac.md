<front>

Your team wants to replace the generic dbt welcome message on the docs site with a guide on how to contact the Data Engineering team. How would you accomplish this?

</front>

---

<back>

Create a Markdown file (e.g., `models/overview.md`) and define a docs block named `__overview__` containing your custom contact info.

Example:
```jinja
{% docs __overview__ %}
# Data Team Contacts
For help, slack #data-help...
{% enddocs %}
```

</back>
