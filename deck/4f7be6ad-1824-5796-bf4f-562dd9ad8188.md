<front>

Which method of the object returned by `run_query` allows you to visualize the result set in the CLI for debugging purposes?

</front>

---

<back>

The `.print_table()` method. 

You typically use it within the `log()` function to print the table structure to the console:
```jinja
{{ log(results.print_table(), info=True) }}
```

</back>
