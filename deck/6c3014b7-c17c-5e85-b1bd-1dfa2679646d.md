<front>

What function signature is required to define a dbt Python model?

</front>

---

<back>

A dbt Python model must be defined by a function named `model` that accepts exactly two arguments: `dbt` and `session`. It must return a final DataFrame.

```python
def model(dbt, session):
    # logic here
    return df
```

</back>
