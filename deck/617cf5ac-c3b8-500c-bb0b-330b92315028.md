<front>

You are setting up a new dbt Core project locally and notice that every time you run `dbt compile`, thousands of new files appear in your `git status`. What is the likely cause?

</front>

---

<back>

The `.gitignore` file is missing or incorrectly configured. You need to add the `target/` directory to your `.gitignore` file to ensure that compiled SQL files and build artifacts generated by dbt are not tracked by git.

</back>
