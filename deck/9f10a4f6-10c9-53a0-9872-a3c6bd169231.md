<front>

A custom maintenance macro is running successfully, but you cannot see the progress messages you added using `{% do log(...) %}` in your terminal. What is the likely cause?

</front>

---

<back>

You likely omitted the `info=true` argument in the log function.

To fix this, change your code to:
```jinja
{% do log("Your message", info=true) %}
```

</back>
