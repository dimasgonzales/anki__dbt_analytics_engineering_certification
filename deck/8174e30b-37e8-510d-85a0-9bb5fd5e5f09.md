<front>

How does dbt determine the lineage for an exposure since exposures do not contain SQL logic?

</front>

---

<back>

dbt determines lineage for exposures via the `depends_on` property in the YAML configuration. You must explicitly list the upstream resources using `ref()` for models and `source()` for sources.

</back>
