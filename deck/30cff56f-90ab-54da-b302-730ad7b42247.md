<front>

You are configuring a deployment job for a project utilizing the dbt Fusion engine. A standard `dbt parse` is not updating the Semantic Layer correctly. How should you adjust the job commands?

</front>

---

<back>

You should replace or append the command with `dbt docs generate`. 

While standard implementations use `dbt parse` to update the manifest, the dbt Fusion engine specifically requires the docs generation command to produce the necessary artifacts.

</back>
