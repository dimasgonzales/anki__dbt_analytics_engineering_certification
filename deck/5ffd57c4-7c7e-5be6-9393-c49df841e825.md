<back>

Use a Jinja if-statement to check the target name:

```sql
{% if target.name == 'prod' %}
    {{ custom_schema_name | trim }}
{% else %}
    {{ default_schema }}_{{ custom_schema_name | trim }}
{% endif %}
```
