<front>

What is the required function signature and return type for a dbt Python model?

</front>

---

<back>

A dbt Python model must define a function named `model` that takes exactly two arguments: `dbt` and `session`. It must return a DataFrame object representing the final dataset to be materialized.

Example:
```python
def model(dbt, session):
    # ...
    return final_df
```

</back>
