<front>

What is the purpose of the `.dbtignore` file in a dbt project?

</front>

---

<back>

The `.dbtignore` file tells dbt to completely ignore specified file patterns or directories. Files matching these patterns are not read, parsed, or detected by dbt, functioning identically to `.gitignore`.

</back>
