<front>

Your compiled SQL contains excessive blank lines around a Jinja `for` loop used to generate columns. How should you modify the syntax to fix this?

</front>

---

<back>

Add whitespace control characters (`-`) to the loop tags to suppress the newlines.

Example:
```sql
select
{%- for col in columns %}
  {{ col }},
{%- endfor %}
...
```

</back>
