<front>

You have a table named `raw_sf_001` in your database, but you want to reference it simply as `customers` in your dbt project using `{{ source('salesforce', 'customers') }}`. How do you configure this in `sources.yml`?

</front>

---

<back>

Use the `identifier` property to map the logical name to the physical table name.

```yaml
tables:
  - name: customers
    identifier: raw_sf_001
```

</back>
