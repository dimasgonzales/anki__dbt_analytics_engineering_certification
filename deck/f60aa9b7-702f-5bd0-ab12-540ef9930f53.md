<front>

You are working in the `marketing` project and need to select data from the `dim_users` model located in the `platform_core` project. How do you write the `ref()` function?

</front>

---

<back>

Use the two-argument `ref()` function, specifying the project name first and the model name second.

```sql
select * from {{ ref('platform_core', 'dim_users') }}
```

</back>
