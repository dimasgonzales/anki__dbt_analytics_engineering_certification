<front>

Do data tests inherit tags defined at the Model level in a YAML configuration?

</front>

---

<back>

No. Tests do **not** inherit tags from the Model, Seed, or Snapshot entity. However, tests **do** inherit tags applied to specific **columns** or **sources**.

</back>
