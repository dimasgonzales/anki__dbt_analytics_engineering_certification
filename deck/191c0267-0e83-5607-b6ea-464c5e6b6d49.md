<front>

You are loading a seed file containing zip codes. dbt is automatically converting '02110' to the integer `2110`. How do you fix this?

</front>

---

<back>

Configure the column type to be a string in `dbt_project.yml` to preserve the leading zero:
```yaml
seeds:
  my_project:
    zip_codes:
      +column_types:
        zip_code: varchar(5)
```

</back>
