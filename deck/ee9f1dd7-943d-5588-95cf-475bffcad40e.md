<front>

How can you access an environment variable in dbt Cloud using Jinja?

</front>

---

<back>

Either:  
`{{ env_var('DBT_MY_ENV') }}`  
or:  
`{{ env_var('DBT_MY_ENV', '<default_value>') }}`
  
  
The first syntax will throw an error if the variable is not defined, while the second will return the given default value if the variable is not defined.

</back>
