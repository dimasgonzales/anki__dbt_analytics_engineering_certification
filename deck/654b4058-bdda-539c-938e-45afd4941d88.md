<front>

You are configuring a new dbt Cloud project via 'Git clone' and receive an error when inputting the repository URL: `https://github.com/dbt-labs/my-repo.git`. How should you fix this?

</front>

---

<back>

Change the URL to the **SSH format**. The error occurs because dbt Cloud strictly enforces SSH for the 'Git clone' method. 

Correct format:
```bash
git@github.com:dbt-labs/my-repo.git
```

</back>
