<front>

Which command is used to compile the project's documentation, catalog, and lineage graph into a static HTML site?

</front>

---

<back>

```bash
dbt docs generate
```

This command queries the information schema of your database to produce the `catalog.json` and combines it with the manifest to build the documentation site.

</back>
