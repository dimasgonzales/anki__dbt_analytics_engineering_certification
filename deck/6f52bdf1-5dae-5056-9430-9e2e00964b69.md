<front>

You are setting up linting for a project and notice syntax errors flagging `{{ config(...) }}` blocks despite valid Jinja usage. How do you fix this in the `.sqlfluff` file?

</front>

---

<back>

Set the `templater` configuration to `dbt`. This instructs SQLFluff to use dbt's compilation context to resolve macros and references prior to applying linting rules.

```ini
[sqlfluff]
templater = dbt
```

</back>
