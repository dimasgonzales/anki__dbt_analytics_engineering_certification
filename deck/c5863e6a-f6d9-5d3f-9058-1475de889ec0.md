<front>

How can you specify the data types of columns in a seed?

</front>

---

<back>

Using the `column_types` config. This taskes a dictionary mapping seed column names to data types (specific to your warehouse/SQL flavor of choice). Example:  

```
column_types:
  country_code: varchar(2)
  country_name: varchar(32)
```

</back>
