<front>

What command is used to verify if source data meets defined SLA thresholds before running transformations?

</front>

---

<back>

The command `dbt source freshness`.

It checks the freshness of sources defined in your YAML files against `warn_after` and `error_after` criteria. It generates a `sources.json` artifact detailing the pass/fail status.

</back>
