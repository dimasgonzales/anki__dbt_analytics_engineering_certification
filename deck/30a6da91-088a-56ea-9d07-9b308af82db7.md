<front>

You need to configure a CI check that validates YAML syntax and project structure as quickly as possible without requiring a database connection. Which command should you use?

</front>

---

<back>

Use `dbt parse`. Unlike `compile` or `run`, `dbt parse` validates the project structure and YAML syntax offline (without connecting to the warehouse), making it the fastest way to check for structural errors.

</back>
