<front>

You need to audit a recent job run to document which models saved compute time by not running. How would you visualize this using the dbt Cloud UI?

</front>

---

<back>

Open the run details, select the **Lineage** tab, and apply the **Latest status** lens. Identify nodes on the graph tagged with **Reused** to see which models utilized existing artifacts.

</back>
