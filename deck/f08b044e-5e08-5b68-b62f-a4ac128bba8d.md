<front>

What type of database error typically indicates that a local seed file has drifted from the warehouse table definition?

</front>

---

<back>

Errors such as `invalid identifier` (Snowflake) or `column... does not exist` (Redshift/Postgres) usually indicate schema drift. This means the CSV contains columns that are not present in the existing target table.

</back>
