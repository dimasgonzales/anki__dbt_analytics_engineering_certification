<front>

You have `fail_fast: true` set in your `dbt_project.yml`, but you define `export DBT_FAIL_FAST=false` in your terminal. When you run `dbt run`, will dbt fail fast?

</front>

---

<back>

**No.**

The Environment Variable (`DBT_FAIL_FAST=false`) takes precedence over the configuration in `dbt_project.yml`. To force it to be true again without changing the env var, you would need to run `dbt run --fail-fast` (CLI overrides Env Var).

</back>
