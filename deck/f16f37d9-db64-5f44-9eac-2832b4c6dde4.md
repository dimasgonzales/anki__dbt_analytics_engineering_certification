<front>

Given the following YAML configuration, write the correct Jinja syntax to reference the `orders` table.

```yaml
sources:
  - name: ecommerce_store
    tables:
      - name: transactions
```

</front>

---

<back>

```sql
{{ source('ecommerce_store', 'transactions') }}
```

The arguments must match the `name` keys provided in the YAML (`ecommerce_store` and `transactions`), regardless of the physical table names in the database.

</back>
