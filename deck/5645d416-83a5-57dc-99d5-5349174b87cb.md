<front>

When determining whether to run a full refresh, which takes precedence: the CLI flag `--full-refresh` or the `full_refresh` configuration in the model file?

</front>

---

<back>

The model configuration takes precedence. If a model config explicitly sets `full_refresh: false`, dbt will ignore the CLI `--full-refresh` flag for that specific model.

</back>
