<front>

How do you reference a specific older version of a model instead of the latest version in your SQL code?

</front>

---

<back>

You use the `v` argument within the `ref()` function to specify the version number.

Example:
```sql
-- Reference version 1 explicitly
select * from {{ ref('fct_orders', v=1) }}
```

</back>
