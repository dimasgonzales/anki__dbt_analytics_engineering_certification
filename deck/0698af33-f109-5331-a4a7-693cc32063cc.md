<front>

You are defining `models/staging/stg_orders.sql` and realize you need the user's email from the raw users table. How should you implement this in dbt?

</front>

---

<back>

Do not join inside the staging model. Instead:
1. Create `stg_orders` (cleaning orders) and `stg_users` (cleaning users) separately.
2. Create a downstream model (intermediate or mart) that joins them:
```sql
select ...
from {{ ref('stg_orders') }}
join {{ ref('stg_users') }} using (user_id)
```

</back>
