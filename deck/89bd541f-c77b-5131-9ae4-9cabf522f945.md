<front>

Which field in `run_results.json` indicates exactly how many records failed a specific data test?

</front>

---

<back>

The `failures` field.

This integer value represents the count of records that violated the test assertion.

Example:
```json
{
  "status": "fail",
  "failures": 25
}
```

</back>
