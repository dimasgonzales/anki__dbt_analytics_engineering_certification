<front>
What **dbt\_project.yml** field is used to specify a custom list of directories where docs blocks are located? What is the default?
</front>

---

<back>

**docs-paths**. By default, dbt will search in *all* resource paths for docs blocks (i.e. the combined list of model-paths, seed-paths, analysis-paths, macro-paths and snapshot-paths). If this option is configured, dbt will *only* look in the specified directories for docs blocks.

</back>
