<front>

You have a Python script using `threading` to invoke `dbtRunner().invoke(["run"])` for three different models simultaneously to save time. The script is failing with unpredictable errors. What is the solution?

</front>

---

<back>

Refactor the script to use **multiprocessing** or an external orchestrator to spawn distinct processes. dbt commands cannot run in parallel threads inside a single process; process isolation is required for concurrent execution.

</back>
