<front>

You have a source table `raw_logs` that should only be enabled when the variable `is_production` is set to true. How would you implement this in the schema.yml?

</front>

---

<back>

Add a config block to the table definition using an inline Jinja expression to check the variable:

```yaml
- name: raw_logs
  config:
    enabled: "{{ var('is_production', false) }}"
```
If the variable is missing or false, the source is disabled.

</back>
