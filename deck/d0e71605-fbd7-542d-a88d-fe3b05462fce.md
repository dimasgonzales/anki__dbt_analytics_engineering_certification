<front>

You need to trigger a run specifically for **incremental** models located inside the **marts/finance** directory. How do you express this in a single CLI command?

</front>

---

<back>

Combine the path selector and the config selector using a comma to enforce intersection.

```bash
dbt run --select "marts.finance,config.materialized:incremental"
```

</back>
