<front>

Your model is failing with a generic SQL syntax error during `dbt run`. How can you view the exact query dbt tried to execute to troubleshoot the syntax manually?

</front>

---

<back>

Inspect the file located in `target/run/{project_name}/models/{model_path}`.

You can copy this SQL and execute it directly in your data warehouse console to pinpoint the specific line causing the syntax error.

</back>
