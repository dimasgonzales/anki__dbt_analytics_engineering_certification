<front>

What two YAML keys are required to correctly configure a Git-based package dependency in `packages.yml`?

</front>

---

<back>

You must use `git` (for the repository URL) and `revision` (for the branch, tag, or commit hash). 

Example:
```yaml
- git: "https://github.com/dbt-labs/dbt-utils.git"
  revision: 1.0.0
```

</back>
