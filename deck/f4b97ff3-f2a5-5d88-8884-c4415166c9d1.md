<front>

You are developing a complex model on a large dataset and want to verify the logic quickly in CI without processing all historical data. How would you accomplish this?

</front>

---

<back>

Run your build command with the `--sample` flag:
```bash
dbt build --sample
```
This applies time-based sampling to inputs, validating the transformation logic efficiently while minimizing warehouse costs.

</back>
