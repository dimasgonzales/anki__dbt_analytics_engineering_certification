<front>

What are the required Jinja tags to start and end a macro definition block in a `.sql` file?

</front>

---

<back>

A macro is defined using the `{% macro macro_name(arguments) %}` tag to start and the `{% endmacro %}` tag to close the definition block. These are placed in `.sql` files typically located in the `macros` directory.

</back>
