<front>

You have successfully built a metric model in your IDE within a Development environment. However, the API is not returning the metric data. What step in the SDLC is likely missing?

</front>

---

<back>

You likely haven't merged the code to the production branch and run a Deployment job. 

The Semantic Layer only serves metrics from **Deployment** environments. You must merge to `main` (or your production branch) and trigger the deployment job to update the manifest for the API.

</back>
