<front>

What is the primary purpose of the `dbt compile` command?

</front>

---

<back>

The `dbt compile` command compiles source code (SQL and Jinja) into executable SQL without running it against the data warehouse. It resolves all refs, macros, and variables, saving the resulting SQL files to the `target/compiled` directory for inspection.

</back>
