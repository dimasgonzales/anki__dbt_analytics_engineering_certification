<front>

You defined `start_date: '2023-01-01'` in the `vars:` block of `dbt_project.yml`. However, when you reference `{{ var('start_date') }}` in a model configuration within the same file, it returns an error or null. Why?

</front>

---

<back>

Variables defined in the `vars:` block are not accessible to the `models:` configuration block within the same `dbt_project.yml` file. To use a variable here, you must supply it via the CLI:

```bash
dbt run --vars '{"start_date": "2023-01-01"}'
```

</back>
