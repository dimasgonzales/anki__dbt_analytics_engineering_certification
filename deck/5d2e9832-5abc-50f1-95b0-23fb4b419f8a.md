<front>

How do you enable the activation of exposure orchestration in a specific dbt Cloud environment?

</front>

---

<back>

You must configure **Environment Variables** in dbt Cloud settings rather than using project YAML files. Specifically, you use the variable `DBT_ACTIVE_EXPOSURES` to turn the feature on.

</back>
