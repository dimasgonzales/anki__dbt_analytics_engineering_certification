<front>
What does the `dbt compile` command do?
</front>

---

<back>

It generates executable SQL from source model, test, and analysis files. The compiled files are placed in your configured target directory. This is for debugging purposes only and is *not* a prerequisite of 
```
dbt run
```

</back>
