What are the two primary functions of the `{{ ref() }}` function in dbt?

</front>

---

<back>

1. **DAG Construction**: It tells dbt which models depend on others to determine the correct execution order.
2. **Environment Management**: It automatically replaces the reference with the correct database and schema for the current environment (e.g., dev vs. prod).
