<front>

You need to integrate dbt logs with a monitoring tool like Datadog or AWS CloudWatch to automatically track error rates and run durations. How should you configure the dbt command?

</front>

---

<back>

Run dbt with the `--log-format json` flag.

Example:
```bash
dbt build --log-format json
```
Monitoring tools can easily ingest and query the structured JSON output compared to unstructured text logs.

</back>
