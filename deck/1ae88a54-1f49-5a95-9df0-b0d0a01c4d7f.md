<front>

Which artifact file does dbt generate after running a freshness check, and what information does it contain?

</front>

---

<back>

`sources.json`. It is generated in the `target/` directory and contains metadata about the freshness execution, including pass/fail status, max loaded timestamps, and calculation times for every source checked.

</back>
