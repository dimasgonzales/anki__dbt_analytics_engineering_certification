<front>

What is the primary purpose of using the `--log-format json` flag in dbt?

</front>

---

<back>

It formats log output as structured JSON objects instead of human-readable text. This enables machine readability, capturing metadata like `invocation_id`, timestamps, and error codes for programmatic parsing.

</back>
