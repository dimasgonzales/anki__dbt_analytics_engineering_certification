<front>

How do you configure a Redshift distribution style within a dbt model file?

</front>

---

<back>

You use the `{{ config() }}` macro at the top of the SQL file. You must explicitly set the `dist` argument to the desired style (or column name) and ensure the model is materialized as a table.

Example:
```sql
{{ config(
  materialized='table',
  dist='user_id'
) }}
```

</back>
