Which dbt command must be run to create the `catalog.json` and `manifest.json` artifacts required to visualize the project structure?

</front>

---

<back>

```bash
dbt docs generate
```
This command inspects the project code and the data warehouse to generate the metadata artifacts needed for the documentation website and lineage graph.
