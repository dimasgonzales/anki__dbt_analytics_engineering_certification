What five primary checks does the `dbt debug` command perform sequentially?

</front>

---

<back>

The `dbt debug` command validates:
1. `dbt_project.yml` validity
2. `profiles.yml` configuration
3. System dependencies (Git, Python)
4. Adapter installation
5. Database connectivity
