<front>

Given the following configuration in a Python model, what will happen during `dbt run`?
```python
def model(dbt, session):
    dbt.config(materialized = 'view')
    # ... logic
```

</front>

---

<back>

The run will **fail**.

dbt Python models do not support `view` materialization. You must change the configuration to `table` or `incremental`.

</back>
