<front>

You have defined a source table with the logical name `users` and the physical identifier `RAW_USERS_V2`. How must you reference this table in your model SQL files?

</front>

---

<back>

You must use the logical name inside the `{{ source() }}` function. Do not reference the physical identifier or the physical schema directly in the SQL.

```sql
-- Correct usage
select * from {{ source('my_source', 'users') }}
```

</back>
