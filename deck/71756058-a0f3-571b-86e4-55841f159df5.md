<front>

What specific artifact must a dbt project generate to allow the dbt Semantic Layer to consume and serve metric definitions?

</front>

---

<back>

The project must generate a `semantic_manifest.json` file. This file is typically located in the `target/` directory and contains the full definitions of the semantic models and metrics required by the Semantic Layer APIs.

</back>
