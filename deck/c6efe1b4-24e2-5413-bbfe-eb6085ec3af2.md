<front>

Which Jinja function allows you to safely pass Git credentials or tokens into `packages.yml` without hardcoding them?

</front>

---

<back>

The `{{ env_var('VARIABLE_NAME') }}` function. 

This allows `packages.yml` to read sensitive values from the system environment variables at runtime.

</back>
