<front>

What is the function of the `query-comment` configuration in `dbt_project.yml`?

</front>

---

<back>

It controls the text or metadata comment prepended (or appended) to SQL queries generated by dbt. This allows users to inject context into data warehouse logs for auditing or debugging.

</back>
