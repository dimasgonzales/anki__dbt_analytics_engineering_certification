<front>

What triggers the dbt Cloud IDE to switch from the default `sqlfmt` formatter to `SQLFluff`?

</front>

---

<back>

The presence of a `.sqlfluff` configuration file in the project root. If this file is detected, the IDE switches from the default opinionated formatter (`sqlfmt`) to the configurable linter/formatter (`SQLFluff`).

</back>
