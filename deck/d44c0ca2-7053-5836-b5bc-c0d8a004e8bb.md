What are the four ways to trigger jobs in dbt cloud?

</front>

---

<back>

1. On a schedule (e.g. cron)
2. via webhooks (e.g. when a PR is created in GitHub)
3. via the dbt Cloud API
4. Manually via the dbt Cloud UI
