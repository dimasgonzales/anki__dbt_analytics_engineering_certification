<front>

You are reviewing code and see: `SELECT {% my_variable %} FROM table`. The model is not compiling as expected. What is the error?

</front>

---

<back>

The developer used the Statement delimiter `{% %}` instead of the Expression delimiter `{{ }}`.

To inject the value of `my_variable` into the SQL string, the correct syntax is:
```sql
SELECT {{ my_variable }} FROM table
```

</back>
