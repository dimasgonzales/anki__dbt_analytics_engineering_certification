<front>

You have updated the values in `seeds/tax_rates.csv`. You need to load the new data and then immediately run all models that depend on this table. What command pattern achieves this?

</front>

---

<back>

You can use the selection syntax with the plus (`+`) operator to identify the seed and its downstream dependencies:

```bash
# Loads the seed and runs downstream models
dbt build --select tax_rates+
```

</back>
