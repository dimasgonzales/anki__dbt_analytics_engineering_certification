<front>

What is the primary purpose of the `dbt-labs/audit_helper` package during a migration project?

</front>

---

<back>

It is used to validate that a refactored dbt model produces identical results to an original legacy table or stored procedure. It provides macros like `compare_relations` to check row counts and compare data row-by-row to ensure parity.

</back>
