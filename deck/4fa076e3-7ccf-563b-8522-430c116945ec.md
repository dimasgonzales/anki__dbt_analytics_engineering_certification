<front>

What are the four built-in generic data tests available in dbt Core?

</front>

---

<back>

The four built-in tests are:
1. **unique**: Verifies values are unique.
2. **not_null**: Verifies values are not null.
3. **accepted_values**: Verifies values match a provided list.
4. **relationships**: Verifies referential integrity (foreign keys) between models.

</back>
