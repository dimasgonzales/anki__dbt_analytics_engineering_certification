<front>

How does configuration inheritance work between source-level and table-level properties in dbt?

</front>

---

<back>

Configurations defined at the source level (like `freshness` or `loaded_at_field`) apply as defaults to all tables under that source. These settings can be overridden by defining specific values within an individual table's configuration block.

</back>
