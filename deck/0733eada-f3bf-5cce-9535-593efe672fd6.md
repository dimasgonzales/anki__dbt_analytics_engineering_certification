<front>

You view your dbt documentation site and see the model lineage and descriptions, but the column data types and table statistics are missing. What is the most likely cause?

</front>

---

<back>

The `catalog.json` file was not generated or failed to populate. This occurs if the database metadata query step fails or is skipped, while `manifest.json` (which provides lineage/descriptions) was successfully compiled.

</back>
