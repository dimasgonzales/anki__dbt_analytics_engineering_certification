<front>

How can you override dbt's automatic data type inference for specific columns in a seed file?

</front>

---

<back>

You must explicitly define the data types using the `+column_types` configuration in your `dbt_project.yml` file.
```yaml
seeds:
  project_name:
    seed_name:
      +column_types:
        column_name: varchar(50)
```

</back>
