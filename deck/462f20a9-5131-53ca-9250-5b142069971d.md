<front>

What **dbt\_project.yml** field is used to specify a custom directory where dbt will write logs? What is the default? How else can this be set?

</front>

---

<back>

**log-path**. The default is `logs`. This setting can be configured in three places (listed in descending order of precedence):  

1. the --log-path CLI flag
2. the DBT\_LOG\_PATH environment variable
3. dbt\_project.yml's log-path setting

</back>
