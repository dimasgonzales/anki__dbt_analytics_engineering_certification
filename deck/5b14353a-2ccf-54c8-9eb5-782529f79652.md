<front>

An auditor requests access logs from 6 months ago for a compliance review, but the Account Admin cannot scroll to that date in the Audit Log UI. How should the Admin retrieve this data?

</front>

---

<back>

The Admin must use the **Export All** feature. Because the UI is limited to 90 days, clicking 'Export All' is required to receive the full history CSV (covering at least 12 months) via email.

</back>
