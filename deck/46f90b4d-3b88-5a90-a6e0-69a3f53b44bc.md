<front>

How is the `{{ this }}` variable primarily used within the logic of an incremental model?

</front>

---

<back>

`{{ this }}` refers to the currently existing target table in the warehouse. It is used in subqueries to dynamically fetch the high watermark (e.g., the most recent timestamp) from the destination table to filter the source query for new data.

</back>
