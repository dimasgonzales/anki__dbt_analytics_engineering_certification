<front>

What is the difference between the audit data retention available in the dbt Cloud UI versus the total retention on Enterprise plans?

</front>

---

<back>

The dbt Cloud UI displays audit events for the last **90 days**. However, Enterprise plans retain data for at least **12 months**, which can be retrieved via the export feature.

</back>
