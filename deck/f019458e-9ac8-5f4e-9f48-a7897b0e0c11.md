<front>

What are three required top-level keys that must be defined in `dbt_project.yml` for a project to be valid?

</front>

---

<back>

A valid `dbt_project.yml` must include:
1. `name`: The project name (snake_case)
2. `version`: The version of the project
3. `profile`: The name of the connection profile used in `profiles.yml`

</back>
