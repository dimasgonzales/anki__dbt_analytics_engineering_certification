<front>

Where is the code for a dbt Python model executed when you run the `dbt run` command?

</front>

---

<back>

The code is executed remotely on the data platform's infrastructure (e.g., the Colab Enterprise executor for BigQuery, a Snowpark warehouse, or a Databricks cluster), not on the local machine where the dbt CLI is running.

</back>
