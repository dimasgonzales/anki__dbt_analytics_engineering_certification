<front>

You need to rebuild the `orders` model and everything downstream of it. However, you must ensure that any other models feeding into those downstream dependencies are also fresh. Which command should you use?

</front>

---

<back>

You should use the `@` operator on the `orders` model.

Command:
```bash
dbt run --select @orders
```
This ensures that `orders`, all its children, and the parents of those children are executed.

</back>
