<front>

You want to translate the CLI command `dbt run --select @tag:nightly` into a formal YAML selector definition. How should you define the graph operator?

</front>

---

<back>

You should use the `childrens_parents: true` key to replicate the `@` operator behavior.

```yaml
definition:
  method: tag
  value: nightly
  childrens_parents: true
```

</back>
