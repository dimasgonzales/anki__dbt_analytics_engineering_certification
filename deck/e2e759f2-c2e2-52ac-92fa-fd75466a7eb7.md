<front>

When configuring quoting for a BigQuery source in dbt, which keys map to the BigQuery concepts of `Project` and `Dataset`?

</front>

---

<back>

**Database** maps to Project, and **Schema** maps to Dataset.

Example:
```yaml
quoting:
  database: true # Quotes the GCP Project ID
  schema: true   # Quotes the BigQuery Dataset name
```

</back>
