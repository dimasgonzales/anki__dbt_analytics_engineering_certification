<front>

What is the precedence order dbt uses when determining the physical table name to compile for a source?

</front>

---

<back>

dbt checks for the `identifier` configuration first. If an `identifier` exists, it is used as the physical table name. If not, dbt defaults to using the table's `name` property.

</back>
