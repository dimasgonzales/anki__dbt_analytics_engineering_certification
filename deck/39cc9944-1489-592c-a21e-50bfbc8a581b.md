<front>

How can you dynamically enable or disable a source table within its YAML definition based on a run-time flag?

</front>

---

<back>

Use the `config` property combined with the `enabled` key and a Jinja `var()` function:

```yaml
tables:
  - name: heavy_table
    config:
      enabled: "{{ var('ingest_heavy_data', false) }}"
```

</back>
