<front>

How does dbt determine the status of a test when `severity` is set to `error` and both `error_if` and `warn_if` are configured?

</front>

---

<back>

dbt evaluates logic in this order:
1. Checks `error_if`. If met, returns **Error**.
2. If not met, checks `warn_if`. If met, returns **Warning**.
3. If neither is met, returns **Pass**.

</back>
