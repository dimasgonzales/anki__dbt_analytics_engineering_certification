<front>

In a dbt Mesh architecture, what requirement must be met for a user to view private models or the full lineage graph of an upstream project?

</front>

---

<back>

The user must have explicit permissions for that specific upstream project. While `public` models are visible in the account-level lineage to all users, viewing `private` models, full model details, or the upstream project's internal lineage requires specific project access.

</back>
