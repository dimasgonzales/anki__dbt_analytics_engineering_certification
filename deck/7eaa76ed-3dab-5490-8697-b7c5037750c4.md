<front>

How do you declare a dependency on an upstream model within a dbt Python model?

</front>

---

<back>

Use the `dbt.ref()` method. This function returns a DataFrame pointing to the upstream table or view and ensures the Python model is correctly placed in the lineage DAG.

```python
df = dbt.ref("upstream_model_name")
```

</back>
