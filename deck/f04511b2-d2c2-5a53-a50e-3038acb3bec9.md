<front>

What network security step is often required to enable dbt Cloud to communicate with a private BI server for lineage integration?

</front>

---

<back>

You must allowlist dbt Cloud's IP addresses. This involves adding the specific IP addresses for your dbt Cloud region to the firewall or access control list of the server hosting the BI tool (e.g., Tableau Server).

</back>
