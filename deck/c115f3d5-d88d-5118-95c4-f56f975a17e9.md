<front>

Which specific dbt command is required to update the metadata used by dbt Explorer and the lineage graph?

</front>

---

<back>

```bash
dbt docs generate
```

While `dbt run` builds the data objects in the warehouse, `dbt docs generate` compiles the catalog and manifest files required to render documentation and lineage metadata.

</back>
