<front>

You have `fail_fast: true` defined in `dbt_project.yml` and `export DBT_FAIL_FAST=false` set in your terminal. If you run `dbt run` without any flags, what is the behavior?

</front>

---

<back>

**Fail fast is FALSE.**

Environment variables take precedence over the `flags` configuration in `dbt_project.yml`. Since no CLI argument was provided to override the environment variable, dbt respects `DBT_FAIL_FAST=false`.

</back>
