<front>

You need to execute the macro `clean_stale_data` and pass the integer `90` to the `days` argument. Write the correct CLI command.

</front>

---

<back>

```bash
dbt run-operation clean_stale_data --args '{days: 90}'
```
The argument passed to `--args` must be a valid YAML dictionary string where the key matches the macro argument.

</back>
