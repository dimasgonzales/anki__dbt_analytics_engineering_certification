<front>

What are the three distinct CLI commands used to remove artifacts, install external packages, and test database connectivity?

</front>

---

<back>

1. `dbt clean`: Deletes the `target/` and `dbt_packages/` directories.
2. `dbt deps`: Installs dependencies listed in `packages.yml`.
3. `dbt debug`: Tests the connection profile and project configuration.

</back>
