<front>

What should a custom materialization return at the end of its execution, and why is this return value important?

</front>

---

<back>

It should return a dictionary containing the list of created relations:
```jinja
{{ return({'relations': [target_relation]}) }}
```
This updates dbt's **internal relation cache**, preventing performance-heavy lookups against the database's `information_schema` for subsequent models.

</back>
