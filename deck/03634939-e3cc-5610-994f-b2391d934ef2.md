<front>

What are the three model access levels available in dbt, and which one is the default?

</front>

---

<back>

The three access levels are `private`, `protected`, and `public`. The default level is **protected**.

1. `private`: Valid only within the same group.
2. `protected`: Valid within the same project or package.
3. `public`: Valid across projects (dbt Mesh).

</back>
