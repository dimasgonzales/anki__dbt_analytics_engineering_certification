<front>

What is a dbt seed and what command is used to load it into the warehouse?

</front>

---

<back>

A dbt seed is a CSV file located in the `seeds` directory of a dbt project, intended for static, small reference data. It is loaded into the data warehouse by running the command:
```bash
dbt seed
```

</back>
