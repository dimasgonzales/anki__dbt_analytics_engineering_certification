<front>

Unlike standard SQL models in BigQuery, what specific cloud storage component is required to execute dbt Python models?

</front>

---

<back>

A **Google Cloud Storage (GCS) bucket**.

dbt uses this bucket to upload the Python code and artifacts to be executed by the BigQuery infrastructure.

</back>
