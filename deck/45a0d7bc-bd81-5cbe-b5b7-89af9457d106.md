<front>

You successfully ran `dbt run` and verified your new model exists in the warehouse, but it is not appearing in the dbt Docs lineage graph. How do you fix this?

</front>

---

<back>

Run `dbt docs generate`. The documentation site displays the state of the project based on static metadata artifacts, not the real-time warehouse state. These artifacts must be regenerated to reflect recent code changes.

</back>
