<front>

You have created a macro named `cents_to_dollars`. You want the generated documentation site to explain what the `column_name` argument does. How would you configure this in YAML?

</front>

---

<back>

Add an `arguments` list under the macro definition in your YAML file.

Example:
```yaml
macros:
  - name: cents_to_dollars
    arguments:
      - name: column_name
        description: The column containing cent values
```

</back>
