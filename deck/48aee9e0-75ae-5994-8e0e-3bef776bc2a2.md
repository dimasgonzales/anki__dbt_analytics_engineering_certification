<front>

What is the primary purpose of the `where` configuration in a dbt test?

</front>

---

<back>

The `where` configuration filters the dataset *before* the test logic is applied. It is primarily used to optimize performance and reduce costs by limiting the test scope (e.g., testing only data from the last 7 days).

</back>
