<front>

You want to access a specific macro available in the `dbt-audit-helper` package to assist with a migration. How do you make this package available to your project?

</front>

---

<back>

Add the package reference and version to your `packages.yml` file, then run `dbt deps` to install it.

```yaml
packages:
  - package: dbt-labs/audit_helper
    version: 0.6.0
```

</back>
