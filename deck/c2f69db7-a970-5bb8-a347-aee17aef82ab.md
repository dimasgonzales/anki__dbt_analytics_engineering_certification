<front>

What are the four built-in generic data tests available in dbt Core?

</front>

---

<back>

The four native generic tests are:
1. `unique`
2. `not_null`
3. `accepted_values`
4. `relationships`

These can be applied directly in YAML configuration files without writing custom SQL.

</back>
