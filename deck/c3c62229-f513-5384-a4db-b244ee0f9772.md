<front>

What is the primary purpose of the `print()` function in dbt Jinja contexts?

</front>

---

<back>

The `print()` function is used to output messages to both standard output (stdout) and the dbt log file. It is primarily used within macros for debugging purposes to inspect variable values or trace execution flow.

Example:
```jinja
{{ print('Variable value: ' ~ my_var) }}
```

</back>
