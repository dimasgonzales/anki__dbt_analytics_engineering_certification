<front>

You need to create a reusable function named `cents_to_dollars` that accepts a column name as an argument. How would you write the definition for this macro?

</front>

---

<back>

You would define the macro with the argument inside the parentheses and encapsulate the logic within the macro tags.

Example:
```sql
{% macro cents_to_dollars(column_name) %}
    ({{ column_name }} / 100)::numeric(16, 2)
{% endmacro %}
```

</back>
