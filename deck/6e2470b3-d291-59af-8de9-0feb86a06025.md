<front>

Which dbt command is responsible for populating the execution state (e.g., 'pass', 'warn', 'error') of source objects?

</front>

---

<back>

The `dbt source freshness` command.

It queries the database to check the `loaded_at_field` against the defined freshness criteria and populates the source status based on the results.

</back>
