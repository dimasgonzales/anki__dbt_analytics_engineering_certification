Documentation by product

Explore the docs by product
---------------------------

[![](/img/icons/dbt-copilot.svg)

#### dbt Copilot

AI-powered assistant that automates code, tests, and documentation in your workflow.](/docs/cloud/dbt-copilot)

[![](/img/icons/vsce.svg)

#### VS Code Extension

This free tool brings the full power of the dbt Fusion engine into your local environment with features like live error detection, lightning-fast parse times, insights and rich lineage all in VS Code or Cursor.](/docs/about-dbt-extension)

[![](/img/icons/deploy.svg)

#### dbt Orchestrator

Every time a job runs, state-aware orchestration automatically determines which models to build by detecting changes in code or data.](/docs/deploy/state-aware-about)

[![](/img/icons/insights.svg)

#### dbt Insights

dbt Insights in dbt empowers users to seamlessly explore and query data with an intuitive, context-rich interface.](/docs/explore/dbt-insights)

[![](/img/icons/canvas.svg)

#### dbt Canvas

dbt Canvas helps you quickly access and transform data through a visual, drag-and-drop experience and with a built-in AI for custom code generation.](/docs/cloud/canvas)

[![](/img/icons/semantic.svg)

#### dbt Semantic Layer

The dbt Semantic Layer eliminates duplicate coding by allowing data teams to define metrics on top of existing models and automatically handling data joins.](/docs/use-dbt-semantic-layer/dbt-sl)

[![](/img/icons/compass.svg)

#### dbt Catalog

Use dbt Catalog to navigate and manage your projects within dbt to help you and other data developers, analysts, and consumers discover and leverage your dbt resources.](/docs/explore/explore-projects)

[![](/img/icons/dashboard.svg)

#### Studio IDE

The dbt integrated development environment (Studio IDE) is a single web-based interface for building, testing, running, and version-controlling dbt projects.](/docs/cloud/dbt-cloud-ide/develop-in-the-cloud#get-started-with-the-cloud-ide)

[![](/img/icons/lineage.svg)

#### dbt Mesh

dbt Mesh is a framework that helps organizations scale their teams and data assets effectively.](/docs/mesh/about-mesh)
